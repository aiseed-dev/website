## 製油所一つで崩壊する農業モデル

現代農業は化学肥料に依存している。この事実は誰もが知っている。
しかし、その化学肥料がどこから来るのかを追跡した人間はほとんどいない。

リン酸肥料を作るには硫酸が必要だ。
硫酸の原料である硫黄は、石油精製の副産物として回収される。
窒素肥料（尿素）は天然ガスから合成される。
食品包装材の原料であるナフサも、石油精製から生まれる。

:::chain
**化学肥料の生産ルート：**
石油精製 → 硫黄（副産物）
→ 硫酸
→ リン鉱石の処理
→ リン酸肥料
天然ガス → 水素 + CO2
→ アンモニア
→ 尿素（窒素肥料）
石油精製 → ナフサ
→ プラスチック
→ 食品包装材
:::

つまり、石油精製施設を物理的に破壊すれば、三つの危機が同時に発生する。
肥料が作れない。食品を包装できない。物流が止まる。

「封鎖」と「破壊」は根本的に異なる。
封鎖は解除すれば翌日から供給が再開する。
破壊された製油所は修復に3〜5年かかる。
現代農業の食料生産能力は、この3〜5年の間、大幅に低下する。

:::highlight
**単一障害点（Single Point of Failure）：**
化学肥料依存の農業モデルは、石油精製という単一障害点を持つ。
ITの世界では、単一障害点を持つシステムは設計ミスとされる。
農業では、この設計ミスが100年以上放置されてきた。
:::

## アメリカの原油では代替できない

「湾岸の油田が止まっても、アメリカはシェールオイルがあるから大丈夫だ」——
これは構造を理解していない反論だ。

:::chain
**原油の種類と硫黄：**
米国シェールオイル（ライトスイート）→ 硫黄含有量 **0.1〜0.5%**
中東サワー原油（アラビアンヘビー等）→ 硫黄含有量 **2〜3%**
硫黄回収量の差 → **4〜30倍**
:::

世界の硫黄生産の大部分は、サワー原油の精製時に脱硫プロセスで回収される副産物だ。
アメリカがいくらシェールオイルを増産しても、
硫黄の供給量は湾岸の精製能力を代替できない。

## アメリカ農業の構造的破綻

「日本の農業は非効率だ、アメリカのように大規模化すべきだ」
——この主張は、アメリカ農業の構造を全く理解していない。

:::chain
**アメリカ農業の実態：**

大規模モノカルチャー → 同じ作物を広大な土地に → 土壌微生物の多様性が消滅
化学肥料への完全依存 → 窒素・リン・カリウムを外部から投入 → 土壌は「培地」に退化
大量の農薬 → 除草剤（グリホサート等）→ 土壌微生物をさらに殺す
大型機械 → 土壌を圧縮 → 水と空気の循環を破壊
灌漑への依存 → オガララ帯水層の枯渇 → 回復に数千年
:::

:::highlight
**表土の喪失——取り返しのつかない損失：**
アメリカの農地は、過去150年で表土の約50%を失った。
表土1インチ（2.5cm）の形成には500〜1,000年かかる。
大規模農業は、数千年かけて蓄積された土壌資本を、
数十年で使い果たしている。
これは採掘であって、農業ではない。
:::

:::highlight
**オガララ帯水層——枯渇する地下水：**
アメリカ中西部の穀倉地帯を支える巨大地下水脈。
年間の汲み上げ量が、自然補給量の数倍〜数十倍。
カンザス州やテキサス州では、すでに水位が30m以上低下。
完全な回復には数千年〜数万年かかる。
アメリカの「食料安全保障」は、枯渇する地下水の上に建っている。
:::

## 企業参入の矛盾

「農業に企業が参入すれば効率化できる」——
この議論も構造を見ていない。

企業の目的は利益の最大化だ。
利益を最大化するには、コストを下げて収穫量を上げる。
コストを下げるには、化学肥料と大型機械を使う。
収穫量を上げるには、モノカルチャーにする。

:::chain
**企業参入の構造的矛盾：**
企業の論理 → 短期利益の最大化
→ 化学肥料＋大型機械＋モノカルチャー
→ 土壌の劣化を加速
→ さらに化学肥料が必要
→ 石油精製への依存が深まる
→ 単一障害点がさらに致命的になる
:::

企業参入は「効率化」ではなく「脆弱化」だ。
短期の利益と引き換えに、長期の食料安全保障を破壊している。

## リジェネラティブ農業——穀物と牛肉はこれでやれる

自然農法が「小規模・多品種」の世界だとすれば、
リジェネラティブ農業（再生型農業）は**大規模な穀物と畜産を自然の仕組みで回す方法**だ。

:::chain
**リジェネラティブ農業の基本構造：**
不耕起 → 土壌構造を壊さない → 菌根菌ネットワークが生きる
被覆作物 → 常に土壌を緑で覆う → 光合成→液体炭素→土壌微生物の好循環
多品種ローテーション → モノカルチャーの脆弱性を排除
家畜の計画的放牧 → 草を食べる→根が再生→土壌に炭素が固定される
:::

### 穀物——被覆作物と不耕起で作れる

慣行農業の穀物生産は、化学肥料＋除草剤＋大型機械のセットだ。
リジェネラティブ農業では、これを被覆作物と不耕起栽培で置き換える。

クローバーやヘアリーベッチを被覆作物として育て、窒素を固定させる。
その中に穀物を直播する。除草剤の代わりに、被覆作物が雑草を抑える。

:::highlight
**Gabe Brownの実証（ノースダコタ州）：**
5,000エーカーの農場で20年以上リジェネラティブ農業を実践。
化学肥料ゼロ、除草剤ゼロ。
土壌有機物が1.7%→6%以上に回復。
収穫量は慣行農業と同等以上。コストは大幅に低い。
利益率は地域の慣行農家を上回っている。
:::

### 牛肉——計画的放牧が土壌を再生する

牛肉生産は環境破壊の代名詞とされる。
しかしそれはフィードロット（穀物肥育）の話だ。

計画的放牧（Adaptive Multi-Paddock Grazing）では、
牛が草を食べ、移動し、草が再生する——このサイクルが土壌を再生する。

:::chain
**計画的放牧の仕組み：**
牛が草を食べる → 根が一部枯死する → 枯死した根が土壌有機物になる
牛が移動する → 草が再生する → 新しい根が伸びる → さらに深くまで炭素が入る
糞と尿 → 微生物のエサになる → 養分循環が回復する
高密度・短期間の放牧 → 草原の自然な撹乱パターンを再現
:::

草原で育つ牛は穀物を食べない。
つまり、穀物を牛に食べさせて非効率に変換する構造がなくなる。
人間が食べられない草を、人間が食べられる肉に変換する——
これが本来の畜産の構造だ。

:::highlight
**リジェネラティブ農業の位置づけ：**
自然農法（福岡正信）→ 小規模・多品種・究極の自立
リジェネラティブ農業 → 大規模な穀物・畜産を自然の仕組みで運営
両者は対立しない。スケールが違うだけで、原理は同じだ。
土壌微生物を殺さない。外部入力に依存しない。光合成を最大化する。
:::

## 自然農法はサバイバル戦略である

福岡正信の四原則——不耕起、無肥料、無農薬、無除草——は、
しばしば「理想論」や「哲学」として語られる。

しかし、前述の構造分析を踏まえれば、
自然農法の本質は全く異なるものとして見える。

:::chain
**自然農法の構造的意味：**
無肥料 = 石油精製への依存ゼロ（単一障害点の排除）
不耕起 = 菌根菌ネットワークの維持（養分供給の自律性）
無農薬 = 化学サプライチェーンへの依存ゼロ
**無除草** = **光合成面積の最大化（Light Farmingの原則そのもの）**
:::

### 無除草——福岡正信の独自性とLight Farmingへの接続

不耕起、無肥料、無農薬は、有機農業やパーマカルチャーとも共通する原則だ。
しかし、**無除草は福岡正信に固有の原則**であり、
他の自然農法や有機農業とは根本的に異なる。

有機農業でも雑草は敵だ。手で抜く。マルチで覆う。
パーマカルチャーでも、デザインによって雑草を管理する。
福岡正信だけが、「草を抜くな」と言った。

なぜか。
Christine Jones博士のLight Farmingが、50年後にその理由を科学的に証明した。

:::chain
**無除草 → Light Farmingの因果連鎖：**
草を抜かない → 土壌が常に緑で覆われる
→ 光合成面積が最大化される
→ 光合成産物（糖）が根から滲出される
→ 土壌微生物が養われる
→ 菌根菌ネットワークが拡大する
→ 液体炭素が土壌に安定的に固定される
→ 土壌の炭素貯蔵量が増加する
→ 土壌構造が改善し、保水力が向上する
:::

:::highlight
**福岡正信の無除草 = Jones博士のLight Farming：**
福岡正信は哲学として「草を抜くな」と言った。
Christine Jones博士は科学として「裸地を作るな」と言った。
50年の時間差があるが、同じ構造を見ている。
多様な植物が常に土壌を覆い、光合成し、根から液体炭素を送り続けること——
これが土壌再生の核心であり、他の自然農法にはない福岡正信の独自性だ。
:::

## 正直な課題——野菜が雑草に負ける

理論は明快だ。しかし、正直に言わなければならないことがある。
無除草での栽培は、実践としては極めて難しい。

:::chain
**最大の課題——野菜が雑草に負ける：**
雑草は強い。野菜は弱い。これが現実。
雑草 → 何千年もその土地に適応してきた。成長が速い。根が深い。
野菜 → 人間が品種改良で作った。肥料と除草を前提に設計されている。
草を抜かない → 雑草が光を奪う → 野菜が育たない
:::

これは「移行期間」の問題ではない。
野菜という植物が、そもそも雑草との競争に弱いという構造的な問題だ。

### 福岡正信の答え——すでに実証されている

福岡正信自身がこの問いに対する答えを、
『無III 自然農法』の中で写真と共に示している。

:::highlight
**写真33：「何もしないで出来るカンピョウや南瓜」**
雑草の中で、カンピョウや南瓜が育っている。
「雑草に勝てる作物」だ。大きな葉を広げ、地面を覆って光を奪い返す。
:::

:::highlight
**写真34：「雪の中の雑草より強い野菜」**
冬、雑草が弱る季節に、野菜が優位に立っている。
「タイミング戦略」の実証だ。
:::

:::highlight
**写真35：「荒地で野草化した大根やカブ」**
最も重要な写真。大根やカブが「野草化」している。
人間の管理なしに、雑草と共存して自生している。
品種改良で失われた「野性」を取り戻した野菜だ。
:::

:::chain
**福岡正信が示した三つの戦略：**
戦略1：雑草に勝てる作物を選ぶ → カンピョウ、南瓜（写真33）
戦略2：雑草が弱い季節に育てる → 冬野菜（写真34）
戦略3：野菜を「野草化」させる → 大根、カブの自生（写真35）
:::

戦略3が最も重要だ。
「雑草に勝てる野菜を選ぶ」のではなく、「野菜を雑草のように強くする」。
数世代で、その土地に適応した「野草化した野菜」が生まれる。
これは品種改良の逆だ。

## 実践的な6つのアプローチ

:::highlight
**1. 雑草に勝てる作物だけを選ぶ**
かぼちゃ・ズッキーニ（大葉で地面を覆う）。トウモロコシ・オクラ（背が高く雑草の上に出る）。サツマイモ（ツルで地表を制圧する）。勝てない野菜は、無除草では作らないと割り切る。
:::

:::highlight
**2. 果樹・多年草に移行する**
果樹は一度根付けば毎年雑草より優位に立てる。福岡正信の果樹園がうまくいったのは、この構造による。
:::

:::highlight
**3. 被覆作物で「雑草の席を先に埋める」**
クローバー、ヘアリーベッチで地面を先に覆い、その中に作物を植える。除草ではない。生態系の設計だ。
:::

:::highlight
**4. タイミングで先行する**
早春、雑草がまだ弱い時期に作物を先行させる。福岡正信の粘土団子もこのタイミング戦略の一つだ。
:::

:::highlight
**5. AIで「勝てる組み合わせ」を蓄積する**
どの作物が、どの雑草に対して、どの季節に勝てるのか。AIによる画像記録と分析で、「この条件ならこの作物が雑草に勝つ」というパターンを蓄積する。
:::

:::highlight
**6. 野草化——野菜を雑草のように強くする**
福岡正信の最終戦略。大根やカブを雑草の中で自然に生えさせ、雑草に負ける個体は淘汰され、勝てる個体だけが種を残す。最も時間はかかるが、最も手間がかからない。究極の無除草だ。
:::

## 今すぐ始めなければならない理由

### 第一の理由——野菜の大量生産が不可能になる

化学肥料の原料は化石資源だ。
天然ガスから窒素肥料を作り、石油精製の副産物である硫黄からリン酸肥料を作る。
化石資源は有限だ。安い油田・ガス田から先に掘り尽くされ、
採掘コストが年々上がっていく。
化学肥料は「なくなる」のではなく、**「高すぎて買えなくなる」**。

穀物と牛肉は、リジェネラティブ農業で化学肥料なしに大規模生産できる。
被覆作物と不耕起で穀物を作り、計画的放牧で牛肉を作る——
これは既にGabe Brownらが実証している。

**しかし、野菜は違う。**

:::highlight
**野菜が抱える構造的問題：**
野菜は雑草に負ける。化学肥料と除草剤なしでは、現行品種の大量栽培が成り立たない。
リジェネラティブ農業は穀物と畜産に有効だが、野菜には直接適用できない。
福岡正信の無除草戦略（雑草に勝てる作物、季節戦略、野草化）は原理として正しいが、
**現代の野菜品種の大半は、化学肥料と除草を前提に品種改良されている**。
化学肥料なしで野菜をどう大量生産するか——この栽培方法自体が確立されていない。
:::

:::chain
**野菜に必要な研究：**
どの品種が化学肥料なしで育つか → 品種選定
どの被覆作物と組み合わせれば雑草を抑えられるか → 栽培体系の設計
野草化にどれだけの世代がかかるか → 長期育種
AIで「勝てる組み合わせ」を蓄積 → データ駆動の栽培設計
**これらは全て、研究から始めなければならない**
:::

穀物と牛肉は方法がある。野菜は方法がない。
方法がないから、今すぐ研究を始めなければならない。
化学肥料が買えなくなってから研究を始めたのでは、
**何年も野菜が作れない時代が来る**。

### 第二の理由——地政学リスクで突然止まる

枯渇を待つまでもなく、地政学リスクで突然止まることがある。
製油所の破壊は仮説ではない。
2026年イラン戦争は現在進行形だ。
ホルムズ海峡の封鎖、製油所への空爆——
化学肥料が手に入らなくなる日は、すでに来ている。

### 第三の理由——土壌の劣化は今も進行している

世界の農地の33%はすでに劣化している。
アメリカの表土は150年で約50%失われた。
オガララ帯水層は枯渇に向かっている。
化石資源が使えるうちに、土壌を再生しなければ、
化石資源がなくなった時に再生する土壌すら残っていない。

:::quote
自然農法は「何もしない」農法ではない。
外部システムの崩壊に対して「何も失わない」農法だ。
リジェネラティブ農業は、大規模農業を自然の仕組みに戻す方法だ。
構造的に正しいことと、実践が容易であることは別だ。
この差を埋めることが、今最も必要な仕事だ。
:::
