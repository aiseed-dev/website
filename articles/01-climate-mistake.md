## 再生可能エネルギーは「エネルギー」しか解決しない

気候変動対策といえば、再生可能エネルギーだ。太陽光、風力、EV、水素。30年間、世界はこの方向に巨額の投資を続けてきた。

しかし2026年のホルムズ危機が突きつけたのは、根本的な問いだ。

**エネルギーを再生可能にしたところで、素材はどうするのか。食料はどうするのか。**

電力を太陽光で作れても、食品を包むプラスチックは作れない。風力で工場を回せても、化学肥料の原料は生まれない。EVが走っても、透析器の膜は製造できない。

エネルギーは文明の一部に過ぎない。再生可能エネルギーは「エネルギー問題」を解決するかもしれないが、「石油依存」は解決しない。

:::highlight
**「石油を燃やさない」と「石油に依存しない」は全く別の問題だ。**
30年間の気候変動対策は、前者しか議論していない。
:::

## 石油は「燃料」ではない——「素材」だ

石油から作られるナフサは、現代文明の物質的基盤を支えている。

:::chain
**石油が「素材」として支えているもの：**

透析器の膜（ポリスルホン）
おむつの吸水材（高吸水性ポリマー）
食品の包装（ポリエチレン、PET、ポリスチレン）
医療器具（PVC）
農業用フィルム（ポリエチレン、ポリオレフィン）
合成繊維（ポリエステル、ナイロン）
建築資材、自動車部品、電子機器の筐体
:::

これらは石油を「燃やして」いるのではない。石油を「素材に変換して」いるのだ。CO2を出さなければ解決するという話ではない。燃やさなくても、素材として必要なのだ。

ホルムズ海峡が閉じた時、日本で最初に危機が顕在化したのはガソリン価格ではなかった。食品包装材の不足、医療用プラスチックの在庫逼迫、農業用フィルムの供給停止。エネルギーではなく、**素材が止まった**。

## 再エネのハードウェア自体が石油に依存している

さらに皮肉なことに、再生可能エネルギーの設備自体が石油化学に依存している。

:::chain
**「脱石油」技術の石油依存：**

太陽光パネル → EVAフィルムとバックシート → **石油化学製品**
風力タービン → ガラス繊維強化プラスチックのブレード → **石油化学製品**
EV → バッテリー部品と車体の軽量化プラスチック → **石油化学製品**
水素 → 電解槽膜と炭素繊維強化プラスチックの貯蔵タンク → **石油化学製品**
:::

再エネの「ハードウェア」が石油に依存している。石油を「燃やさない」社会を作るためのインフラが、石油化学なしには作れない。

:::highlight
**再生可能エネルギーは石油の代替ではない。石油の上に乗っている追加の層だ。**
土台を抜いたら、上に乗っているものも倒れる。
:::

## 合成肥料——80億人の食を支える石油依存

エネルギーと素材に加えて、もう一つの致命的な依存がある。食料だ。

天然ガスから作られる合成肥料（ハーバー・ボッシュ法）が、世界の食料生産の約半分を支えている。再生可能エネルギーでこの合成肥料を代替することはできない。

ではハーバー・ボッシュ法が実用化された1913年以前、植物はどうやって窒素を得ていたか。

**土壌中の微生物がやっていた。**

:::chain
**数億年の生命ネットワーク：**

根粒菌 → マメ科植物の根に共生 → 大気中の窒素を固定
菌根菌 → 植物の根と土壌を繋ぐ → リンやミネラルを運ぶ
無数の微生物 → 有機物を分解 → 養分を循環させる
土壌1グラムに推定**10億**の微生物
数億年かけて構築されたネットワークが、地球の全ての陸上生態系を支えてきた
:::

人類はわずか100年で、このシステムを合成肥料に置き換えた。そしてその合成肥料の原料が、ホルムズ海峡の向こう側にある。

## 土壌の自給力を壊した

合成肥料の大量投入は、土壌微生物そのものを弱体化させた。

:::chain
**依存の悪循環：**

窒素肥料が豊富 → 植物は菌根菌との共生を「必要としなくなる」
→ 共生関係が退化
→ 微生物多様性が低下
→ 土壌の自給力が失われる
→ 作物はさらに合成肥料を必要とする
→ さらに微生物が弱る →（負のフィードバックループ）
:::

そしてその合成肥料の原料が、ホルムズ海峡の向こう側にある。海峡が閉じたとき、自給力を失った土壌は、自力で作物を養えない。

:::highlight
**硫黄トラップ——もう一つの構造的脆弱性：**
リン酸肥料を作るには硫酸が必要だ。
硫酸の原料である硫黄は、石油精製の副産物として回収される。
世界の硫黄生産の約90%が石油・天然ガス精製の副産物。
米国のシェールオイル（ライトスイート原油）は硫黄含有量が0.1〜0.5%。
中東のサワー原油は硫黄含有量が2〜3%。
「エネルギー独立」は「素材独立」を意味しない。
製油所が止まれば、核融合があっても肥料は作れない。
:::

## 気候変動対策の3つの盲点

現在の気候変動対策は「CO2排出量の削減」に集約されている。しかしホルムズ危機を通じて、3つの重大な盲点が見えた。

### 盲点①：素材としての石油

先に述べた通り、石油は燃料だけではなく素材だ。食品包装、医療器具、農業資材——これらの代替なしにCO2だけ減らしても、文明の物質的基盤は石油に依存したままだ。再エネのハードウェアすら石油化学製品で作られている。

### 盲点②：肥料と食料の構造的矛盾

:::chain
**CO2削減と食料生産のトレードオフ：**

天然ガスの使用を減らす → 窒素肥料の生産コストが上がる
合成肥料なしでは → 世界人口の約半分を養えないとされる
有機農業への全面転換 → 慣行農業と比べて収穫量が30〜50%減少するとされる
CO2を減らすことと、80億人を食べさせることの間に、**構造的なトレードオフ**がある
:::

しかしこの問題が気候変動の議論で正面から扱われることは稀だ。

### 盲点③：土壌が解決策そのものであること

最大の盲点は、**土壌と生態系がCO2問題の解決策である**という認識の欠如だ。

:::highlight
**土壌は最大の炭素貯蔵庫：**
世界の土壌には大気中の**2〜3倍**の炭素が蓄積されている。
健全な土壌は最大の炭素貯蔵庫であり、最も安価なカーボンキャプチャー装置だ。
土壌微生物の活動を回復させれば、有機物が安定した形で土壌中に蓄積される。
高価な工業的CO2回収装置を建設するよりも、
劣化した農地の土壌を回復させる方が、はるかにコスト効率が高い。
:::

30年間、この順序が逆だった。

:::compare
| 項目 | 現在の気候変動対策 | 生態系中心の対策 |
| --- | --- | --- |
| 焦点 | CO2排出削減 | 土壌・生態系の回復 |
| 手段 | 再エネ、EV、カーボンプライシング | 土壌再生、生物多様性回復 |
| 石油との関係 | 「燃やさない」 | 「依存しない」 |
| 食料問題 | 議論の外 | 中核に統合 |
| コスト | 兆ドル規模の設備投資 | 既存の農地で実施可能 |
| 炭素固定 | 工業的CCS（高コスト） | 土壌微生物（ほぼ無料） |
:::

## 生態系中心の気候変動対策へ

提案したいのは、気候変動対策のパラダイムを「CO2削減」から**「生態系の回復」**へ転換することだ。

### ① 土壌回復＝カーボンキャプチャー

被覆作物（カバークロップ）の導入、堆肥の活用、多様な作物のローテーション、マメ科植物の戦略的配置。これらは新技術ではない。伝統的農法の知恵の再評価だ。

Christine Jones博士のLight Farmingが示すように、生きた根からの液体炭素（Liquid Carbon）は、地上バイオマスの約5倍の効率で土壌に安定的に固定される。

:::chain
**Light Farmingの炭素固定メカニズム：**

太陽光 → 光合成 → 植物が糖を作る
糖 → 根から土壌へ滲出（Liquid Carbon）
Liquid Carbon → 土壌微生物のエネルギー源
土壌微生物 → 植物に必要なミネラルを可溶化して供給
植物 → さらに光合成 →（正のフィードバックループ）
**土壌微生物が回復すれば、炭素が土壌に固定され、合成肥料への依存も減る。**
**CO2削減と食料安全保障が同時に達成される。**
:::

### ② 生物多様性の回復＝耐障害性

生物多様性の本質は「システムの耐障害性」だ。

:::chain
**多様性が危機耐性を生む：**

多様な微生物がいれば → 一つの種が減っても他が補う
多様な作物があれば → 一つの病害で全滅しない
モノカルチャーから多品種栽培への転換
→ 一つの海峡が閉じても崩壊しない食料供給を実現する
:::

効率を犠牲にするのではない。**効率の定義を変える**。「短期的な収穫量/コスト」から「危機耐性を含めた長期的な持続可能性」へ。

### ③ 素材の脱石油

プラスチックをバイオマスやCO2由来の素材で代替する技術開発を、エネルギー転換と同等の優先度で推進すべきだ。特に食品包装と医療用プラスチックは、人命と食料流通に直結する。「包装がなくて出荷できない」事態を二度と起こさないために。

:::highlight
**備蓄の非対称が「見えていなかった証拠」：**
石油備蓄 → **254日分**
ナフサ備蓄 → **20日分**
肥料備蓄 → **ゼロ**
この非対称自体が、「エネルギーしか見ていなかった」証拠だ。
:::

### ④ 水循環の回復

気候変動で最も人を殺すのは気温上昇そのものではなく、**洪水と干ばつ**だ。

:::chain
**生態系が水を制御する：**

健全な森林 → 雨水を保持しゆっくり放出する
健全な土壌 → 水を吸収し地下水を涵養する
生態系の回復 → 水の問題に直接対処する
:::

## 火星には土壌がない、AIは土壌を作れない

イーロン・マスクは人類の未来は火星にあると言う。しかし火星には土壌がない。微生物がいない。数億年かけて構築された生態系が存在しない。

火星で食料を育てるには、土壌微生物の全ての機能を工業的に再現しなければならない。しかしその工業的システムが、地球上でホルムズ海峡一つで崩壊しかけている。地球ですら生態系を維持できない人類が、火星で一から作れるはずがない。

AIについても同じだ。AIは強力なツールだが、ナフサを生産することも、土壌微生物を培養することもできない。土壌1グラムの中の10億の微生物の相互作用ネットワークを、AIが設計することは原理的に不可能だ。生態系は「設計」されたのではなく「進化」した。

:::highlight
**優先順位の逆転：**
火星移住に莫大な資金と技術を注ぐより、地球の土壌を回復させる方が、はるかに多くの命を救う。
AIに全てを任せるより、土壌微生物に働いてもらう方が、はるかに持続可能だ。
:::

## 福岡正信はすでに答えを出していた

1940年代、福岡正信は自然農法を始めた。不耕起、無肥料、無農薬、無除草。

:::chain
**福岡正信の四原則の構造的意味：**

無肥料 = 石油・天然ガスへの依存ゼロ（サプライチェーンに縛られない）
不耕起 = 土壌構造を壊さない（微生物ネットワークが生きる）
無農薬 = 生態系を殺さない（自然の防御力を活かす）
**無除草** = **光合成面積の最大化（Light Farmingの原理そのもの）**
:::

福岡正信の「草を抜くな」は、Christine Jones博士のLight Farmingの第一原則——「緑は良い、一年中緑はさらに良い（Green is good. Year-long green is even better.）」——と完全に一致する。

福岡正信は哲学として「草を抜くな」と言った。
Christine Jones博士は科学として「裸地を作るな」と言った。
80年の時間差があるが、同じ構造を見ている。

:::quote
気候変動の答えは、テクノロジーの中にはない。
土壌の中にある。
福岡正信とChristine Jones博士が、それぞれ実践と科学で証明した。
自然とともにあれば、生きられる。
自然から離れるほど、生きるコストは上がる。
:::
