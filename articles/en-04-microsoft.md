## The Six Taxes

Microsoft's business model is not selling software.
It is collecting **six layers of tax** from every knowledge worker on earth.

:::chain
**Tax 1: OS Tax** → Windows. The base platform you must use.
**Tax 2: Productivity Tax** → Microsoft 365 (Word, Excel, PowerPoint). You pay monthly.
**Tax 3: Communication Tax** → Teams. Your organization's communication trapped in Microsoft.
**Tax 4: Cloud Tax** → Azure. Your infrastructure locked to Microsoft.
**Tax 5: Developer Tax** → GitHub + VS Code. Where you write and store code.
**Tax 6: AI Tax** → Copilot. $30/user/month ON TOP of Microsoft 365.
:::

Each tax builds on the previous one. You buy Windows, so you buy Office.
You buy Office, so you use Teams. You use Teams, so you need Azure.
You use VS Code, so you add GitHub. You have all of these, so you add Copilot.

The total extraction per knowledge worker: **hundreds of dollars per month**.

## Copilot vs Claude — Linguistic vs Structural Thinking

The critical question is not "which AI is better."
It is "what kind of thinking does each AI represent?"

:::compare
| Capability | Copilot (Microsoft) | Claude (Anthropic) |
|-----------|-------------------|-------------------|
| Core strength | Document processing, summaries | Structural analysis, pattern recognition |
| Thinking model | Linguistic (manipulates words) | Structural (traces production routes) |
| Integration | Locked to Microsoft products | Independent, works anywhere |
| Pricing | $30/month added to existing subscription | Standalone subscription |
| Best for | Making existing office work faster | Seeing connections across fields |
| Code generation | VS Code autocomplete | Claude Code: full project understanding |
| Who benefits most | People who write documents | People who see structure |
:::

Copilot makes you faster at writing documents.
Claude helps you understand **why** those documents should exist.

This is not a feature comparison. It is a **paradigm difference**.

## The Collapse Sequence

When structural thinkers switch from Copilot to Claude,
the six-tax structure begins to unravel — from the top.

:::chain
**Stage 1:** AI Tax peels off → Structural thinkers choose Claude over Copilot
**Stage 2:** Developer Tax peels off → Claude Code replaces VS Code + GitHub Copilot
**Stage 3:** Productivity Tax peels off → AI-generated documents replace Office templates
**Stage 4:** Communication Tax peels off → AI-mediated communication replaces Teams
**Stage 5:** Cloud Tax peels off → Cloud-agnostic AI reduces Azure dependency
**Stage 6:** OS Tax peels off → When applications are AI-native, the OS becomes irrelevant
:::

Each tax depends on the others for justification.
When one falls, the rationale for the rest weakens.
When the AI tax — the newest and most visible — proves inferior to alternatives,
the entire bundle is questioned.

## Why Bundling Collapses

:::highlight
**The bundling paradox:**

Bundling works when each component is "good enough" and switching costs are high.
AI destroys both conditions:
- AI quality varies enormously between providers (Claude vs Copilot is not close)
- AI tools are cloud-based with zero switching cost (open a browser, start using Claude)

When the best tool is NOT in the bundle,
the bundle's value proposition collapses entirely.
:::

Microsoft built an empire on "good enough in every category, bundled together."
But "good enough" AI is not sufficient.
Structural thinking requires the **best** AI, not the most convenient.

## The Structural Parallel

:::chain
**The same pattern across domains:**

Microsoft: Bundled six taxes → Users pay for things they don't need
Chemical agriculture: Bundled inputs (fertilizer + pesticide + machinery) → Farmers depend on supply chains
Superpower military: Bundled forces (army + navy + air force) → Enormous cost, vulnerable to asymmetry

Microsoft alternative: Claude (pay only for what you use)
Agriculture alternative: Natural farming (no external inputs)
Military alternative: Drone + AI (focused, cost-effective)

**Unbundling wins in every domain.**
:::

:::quote
Microsoft's empire was built by bundling.
AI will unbundle it.
When thinking tools replace product suites,
all six taxes disappear — not one by one, but in cascade.
The bundle is the vulnerability.
:::
