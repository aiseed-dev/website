ホルムズ海峡が閉まると、食料が消える

イラン戦争が日本の食卓を直撃する構造

2026.04.05

構造分析ノート

何が起きているか

2026年2月28日、米国とイスラエルが「オペレーション・エピック・フューリー」としてイランへの共同軍事攻撃を開始した。最高指導者ハーメネイーが殺害された。

イランは報復として、以下を実行した：

  • ホルムズ海峡の封鎖宣言（3月4日）——海峡を通過する船舶への攻撃を開始。3月12日までに商船21隻が攻撃された
  • ホルムズ海峡への機雷敷設——米軍は機雷敷設艦16隻を撃沈したが（3月10日）、十数個の機雷が敷設された。イランはなお80〜90%の小型艦艇と敷設能力を保持している
  • 湾岸の石油施設への攻撃——サウジアラビアのシャイバ油田への連日のドローン攻撃、バーレーン・UAE・クウェート・オマーンの製油所・燃料貯蔵施設への攻撃。4月3〜4日にはクウェート最大の製油所が炎上、サウジのヤンブーSAMREF製油所がドローンで被弾
  • 淡水化設備への攻撃——バーレーンの淡水化プラントがドローンで損傷（3月8日）、クウェートの発電・淡水化プラントがミサイルで被弾し作業員1名死亡（3月30日）、UAE・バーレーンの施設も間接被害

トランプ大統領の発言

  • 3月22日：「48時間以内にホルムズ海峡を完全に開かなければ、イラン最大の発電所から破壊する」と威嚇 (CBS News)
  • 3月23日：「非常に生産的な対話」があったとして攻撃を5日間延期 (日本経済新聞)
  • 3月26日：攻撃期限を4月6日午後8時（米東部時間）まで再延期 (Al Jazeera)
  • 3月30日：「合意に至らなければ、イランの発電所、油井、ハーグ島、そして淡水化プラントも全て吹き飛ばす」とTruth Socialに投稿 (Al Jazeera)。アムネスティ・インターナショナルは「戦争犯罪の威嚇」と非難 (Amnesty International)
  • 4月1日：国民向けテレビ演説で「戦争は完了に近づいている」「今後2〜3週間で極めて激しく攻撃する」「石器時代に戻す」と発言 (CNBC, Al Jazeera)
  • 4月3日：「米国はイランの残りの破壊をまだ始めてすらいない。次は橋、その次は発電所だ！」とTruth Socialに投稿 (CNBC)
  • 4月4日：「48時間以内に合意するか海峡を開け。さもなくば地獄が降り注ぐ」（"Time is running out — 48 hours before all Hell will reign down on them"）(Axios)

パキスタン、トルコ、エジプトによる間接交渉が行われているが、イランは一時停戦を拒否し、恒久的な戦争終結と再攻撃しない保証を要求している。4月6日の期限が迫っている。

ブレント原油は2月28日以降55%以上上昇し、126ドル/バレルに達した。国際エネルギー機関（IEA）は「史上最大のエネルギー安全保障上の危機」と評した (Economic impact — Wikipedia)。

多くの人はこれを「中東の戦争」として見ている。

違う。これは日本の食料危機だ。

ホルムズ海峡が閉まると何が止まるか

日本の原油輸入の約9割は中東からだ。そのほぼ全てがホルムズ海峡を通過する。

ホルムズ海峡封鎖 → 原油輸入停止 → 製油所の原料がなくなる → ナフサ・ガソリン・灯油・軽油　全て停止

第2章「化石資源と素材」で説明した通り、原油の蒸留は「選べない」。ガソリンだけ作ることも、ナフサだけ作ることもできない。原油が来なければ、全てが止まる。

ナフサが止まると肥料が止まる

ナフサは石油化学の原料だ。プラスチック、合成繊維、医薬品——現代文明の素材は全てナフサから始まる。

そしてナフサ（または天然ガス）から作られるものの中に、アンモニアがある。

ナフサ / 天然ガス → 水素 → アンモニア → 窒素肥料
窒素肥料 → 世界の食料生産の約50%を支えている

日本は肥料原料もほぼ全量輸入に依存している。

  • 窒素肥料の原料：天然ガス / ナフサ（中東依存）
  • リン酸肥料の原料：リン鉱石（中国・モロッコから輸入）
  • カリウム肥料の原料：カリ鉱石（カナダ・ロシアから輸入）

ホルムズ海峡の封鎖は、窒素肥料の原料供給を直撃する。

ホルムズ海峡封鎖 → 原油・天然ガス停止 → ナフサ停止 → アンモニア停止 → 窒素肥料停止

これは「ガソリンが高くなる」問題ではない。食料生産が止まる問題だ。

機雷封鎖は簡単には解除できない

「ホルムズ海峡の封鎖は一時的だ」と考える人がいる。甘い。

機雷の掃海は極めて時間がかかる作業だ。1隻ずつ、1海里ずつ、機雷を探知して除去していく。戦時中は掃海艦自体が攻撃対象になる。イランが追加で機雷を撒き続ければ、掃海作業は終わらない。

封鎖の解除には数ヶ月から年単位の時間がかかる。

湾岸施設への攻撃——復旧も封鎖で阻まれる

さらに深刻なのは、イランが湾岸の石油施設を攻撃していることだ。

破壊された精製施設の復旧には3〜5年かかる。しかし問題はそれだけではない。海峡が封鎖されている限り、復旧に必要な資材や機材も搬入できない。封鎖と施設破壊は互いを悪化させる。

淡水化設備の破壊——人が住めなくなる

湾岸諸国の致命的な弱点は、飲料水を海水淡水化に依存していることだ。

すでにバーレーンの淡水化プラントがドローンで損傷し（3月8日）、クウェートの発電・淡水化プラントがミサイルで被弾して作業員1名が死亡している（3月30日）。4月3日にもクウェートの淡水化プラントが再び攻撃された。

湾岸6カ国は合計6,200万人の飲料水を淡水化に依存している。淡水化設備が破壊されると：

淡水化設備破壊 → 飲料水がなくなる → 人が住めなくなる → 施設の管理・復旧ができない → 石油生産の再開が不可能に

施設が壊れても、人がいれば直せる。水がなければ、人がいられない。

淡水化設備の破壊は、石油施設の破壊より深刻だ。石油施設は直せる。しかし直す人間が住めなければ、何も始まらない。復旧は数年ではなく、十年単位になる可能性がある。

日本の脆弱性

日本の数字を並べる。

  • 原油の中東依存度：約90%
  • ナフサの大半を輸入に依存
  • 肥料原料：ほぼ全量輸入
  • 食料自給率：38%（カロリーベース）

この構造は、ホルムズ海峡という一つのチョークポイントに全てが集中していることを意味する。

日本の食料は、ホルムズ海峡に依存している。

中東の戦争は、日本の食卓の問題だ。

硫黄も止まる——リン酸肥料の危機

見落とされている事実がある。湾岸諸国は**世界の硫黄供給の約45%**を占めている。硫黄は石油精製の副産物だ。

第11章「規制の再設計」で分析した通り、リン酸肥料は硫酸なしに作れない。硫酸の原料は硫黄だ。

ホルムズ海峡封鎖 → 湾岸からの硫黄輸出停止（世界の45%）→ 硫酸不足 → リン酸肥料の生産に影響

窒素肥料だけではない。リン酸肥料も同時に危機に陥る。第11章で警告した「ピーク・サルファー」が、脱炭素ではなく戦争によって現実化した。

構造分析シリーズとの接続

第1章で指摘した通り、化石資源の枯渇は気候変動より先に来る。

第2章で示した通り、原油の蒸留は選べない。止まれば全てが止まる。

第3章で分析した通り、化学肥料が消えれば慣行農業は成り立たない。

第11章で述べた通り、硫黄供給が止まればリン酸肥料も止まる——ピーク・サルファー。

イラン戦争は、この構造を今この瞬間に現実にしている。

化石資源の枯渇は数十年後に来ると言われていた。しかしホルムズ海峡の封鎖は2026年3月4日に起きた。構造分析シリーズが描いた未来が、戦争という形で先に来た。

自然農法だけが影響を受けない

化学肥料に依存しない農業だけが、この構造的リスクから自由だ。

福岡正信の自然農法、Christine Jones博士のLight Farming。不耕起、無肥料、無農薬。土壌微生物との共生による窒素固定。

ホルムズ海峡が閉まっても、土壌微生物は止まらない。

構造分析の結論は常に同じ場所に収束する。

自然から離れるほど、生きるコストは上がる。イラン戦争は、そのコストが一瞬で跳ね上がることを証明した。
