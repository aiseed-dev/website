## あなたの生活のほぼ全てが石油でできている

朝起きる。ポリエステルのシーツから体を起こす。
プラスチックの歯ブラシで歯を磨く。
ナイロンの服を着る。合成ゴムの靴を履く。
スマートフォンのポリカーボネート筐体を手に取る。
アスファルトの道路を歩く。
食品をポリエチレンの袋から取り出す。
ポリプロピレンの容器に入った弁当を食べる。

この日常のどこに「石油」があるか。全部だ。

:::chain
**石油から作られる日常の素材：**
ナフサ → エチレン → ポリエチレン（レジ袋、ラップ、容器）
ナフサ → プロピレン → ポリプロピレン（食品容器、自動車部品）
ナフサ → ベンゼン → ナイロン、ポリウレタン（衣類、断熱材）
ナフサ → スチレン → ポリスチレン（発泡スチロール、食品トレー）
ナフサ → PET（ペットボトル、ポリエステル繊維）
ナフサ → 塩ビ（配管、窓枠、電線被覆）
重質留分 → アスファルト（道路）
重質留分 → 潤滑油（あらゆる機械の動作）
副産物 → パラフィン（ろうそく、化粧品、医薬品コーティング）
:::

プラスチックは「環境に悪い使い捨て製品」として語られることが多い。
しかし、それは問題の表面にすぎない。
本当の問題は、現代文明のインフラそのものが化石資源の素材機能に依存していることだ。

## プラスチック問題の本質は「代替不能性」

「プラスチックをやめよう」「脱プラスチック」——
この議論は、プラスチックを「選択肢の一つ」として捉えている。
紙に変えればいい。竹に変えればいい。ガラスに変えればいい。

しかし、構造的に見れば、プラスチックが代替不能な用途が大量にある。

:::compare
| 用途 | なぜプラスチックなのか | 代替の現実 |
| --- | --- | --- |
| 医療器具（注射器、チューブ、血液バッグ） | 滅菌可能、軽量、使い捨てで感染防止 | ガラスは重く割れる。金属は高コスト。現実的な代替なし |
| 食品包装 | 気密性、軽量、低コスト、食品との非反応性 | 紙は水に弱い。ガラスは重く輸送コスト増。部分的代替のみ |
| 電線被覆 | 絶縁性、柔軟性、耐候性 | ゴムは劣化が速い。代替なし |
| 水道管・下水管 | 腐食しない、軽量、長寿命 | 金属管は腐食する。陶管は重く脆い。大規模代替は非現実的 |
| 自動車部品 | 軽量化（燃費向上）、成形の自由度 | 金属で全て作ると重量・コスト増大 |
| 半導体のフォトレジスト | 微細パターン形成に不可欠 | 代替なし。半導体製造が停止する |
:::

「脱プラスチック」は、レジ袋やストローの話をしている。
しかし、医療器具、電線、水道管、半導体のプラスチックをやめたら、
文明が止まる。

:::highlight
**プラスチック問題の構造：**
表面の問題：使い捨てプラスチックの環境汚染（レジ袋、ストロー）
本質の問題：現代文明のインフラがプラスチックなしでは機能しない
見えていない問題：そのプラスチックの原料（ナフサ）は石油精製から来る
:::

## 大企業が存在できるのは化石資源があるから

現代の大企業——Amazon、Apple、Toyota、Walmart——は、
化石資源のインフラの上に成り立っている。

:::chain
**大企業を支えるインフラの化石資源依存：**
**物流：**軽油（トラック・船舶）+ アスファルト（道路）+ プラスチック（梱包材）
**製造：**ナフサ（部品素材）+ 潤滑油（機械動作）+ 合成ゴム（シール・ベルト）
**通信：**塩ビ（電線被覆）+ フォトレジスト（半導体）+ 光ファイバー被覆
**食品：**ポリエチレン（包装）+ 硫黄（肥料）+ ナフサ（農薬原料）
**建設：**アスファルト + 塩ビ配管 + 断熱材（ポリウレタン）+ 塗料
:::

Amazonの翌日配送は、アスファルトの道路とプラスチックの梱包材がなければ成立しない。
Appleの製品は、半導体のフォトレジストとポリカーボネートの筐体がなければ作れない。
Toyotaの車は、合成ゴムのシールと潤滑油がなければ動かない。

大企業は「テクノロジー企業」や「製造企業」を自称するが、
構造的には「化石資源の加工業」だ。
化石資源の素材機能がなくなれば、これらの企業のビジネスモデルは崩壊する。

## 全車EV化は無謀である

「全ての自動車を電気自動車にすれば、石油はいらなくなる」——
この発想が、素材の構造を完全に無視している。

### 精製すれば燃料は出てくる。ならば使え。

ナフサのために原油を精製すれば、ガソリンも軽油も**必ず**同時に出てくる。

:::chain
**精製の構造的事実：**
プラスチック、医療器具、肥料原料にナフサが必要
→ ナフサは原油精製でしか得られない
→ 原油を精製すれば、ガソリン・軽油・重油が同時に出てくる
→ 素材が必要な限り、石油精製は止められない
→ 精製が止められないなら、**ガソリンも軽油も必ず出てくる**
→ 出てくる燃料は使うしかない。捨てることすらできない。
:::

### 軽油——電化が不可能な領域

軽油は、使う側も電化できない。

:::highlight
**電化が極めて困難な領域：**
大型トラック（長距離輸送）→ バッテリーの重量で積載量が激減。充電時間が物流を止める。
船舶 → 大型コンテナ船の電化は物理的に不可能。重油・軽油が唯一の現実解。
建設機械 → 高負荷・長時間稼働。電動化は小型機のみ。
農業機械 → 圃場に充電インフラはない。軽油で動くしかない。
:::

軽油は二重の理由で消えない——
**使う側が電化できない**。そして**精製すれば必ず出てくる**。

### ガソリン——ハイブリッド車を残す合理性

ガソリンも精製すれば出てくる。出てくるなら使うべきだ。

乗用車を全てEVにすれば、出てくるガソリンの行き場がなくなる。
しかしガソリンは危険物だ。捨てることはできない。
行き場のない燃料が溜まり続ける——これは構造的な破綻だ。

:::highlight
**現在の精製構造の中でハイブリッド車が合理的な理由：**
ナフサのために精製する → ガソリンが出てくる
→ ハイブリッド車がそのガソリンを高効率で使う
→ EVよりガソリン消費は多いが、精製の副産物を無駄にしない
→ バッテリーの使用量も少ない（リチウム・コバルトの資源制約を緩和）
→ 精製・素材・エネルギーの全体最適
:::

全車EV化は、精製で必ず出てくるガソリンの行き場を消し、
その上で別に電気を作って走る——**エネルギーの二重投資**だ。
しかもガソリンは捨てられない。行き場のない危険物が溜まるだけだ。
**現在のサプライチェーン下では**、ハイブリッド車は出てくるガソリンを
使い切る合理的な選択である。

> ただしこれは「2026 年時点の精製構造」を前提とした話である。
> **20 年・30 年スパンで自動車がどうあるべきかは、本章では答えを
> 出さない**。後段で扱う。

:::highlight
**構造的事実：**
現在、石油精製のコストは86%のエネルギー用途（ガソリン・軽油）の売上で賄われている。
残り14%の化学原料（ナフサなど）は、いわば「ついで」に安く供給されている。
燃料需要が大幅に減れば、素材用途のためだけに石油を精製することになる。
精製コストは現在の何倍にもなる。
ナフサの価格が跳ね上がり、プラスチック、医療器具、肥料——全てが高騰する。
:::

問題は「ガソリン車かEVか」ではない。
**素材の代替（バイオ素材）がどこまで進んでいるか**だ。
バイオ素材がナフサを代替できるようになって初めて、
石油精製を縮小する選択肢が生まれる。順序が逆なのだ。

:::quote
今までの気候変動対策がいかに間違っていたかが、ここで見えてくる。
「石油を燃やすな」→ EV化、再エネ推進——これが主流の気候変動対策だった。
しかし、素材が化石原料に依存している限り、精製は止められない。
精製すれば燃料は出てくる。出てくる燃料は捨てられない。使うしかない。
なのに別に電気を作って走る——二重投資。
CO2排出だけを見て、素材の構造を見なかった。
エネルギーだけを見て、精製プロセスの全体像を見なかった。
気候変動対策は「石油を燃やすな」ではなく、
**「石油に素材を依存する構造を変えろ」**から始めるべきだったのだ。
:::

## 化石資源は有限である——避けられない枯渇

ここまで、精製の構造を見てきた。
素材が必要な限り精製は止められない。燃料は出てくる。使うしかない。
これが現在の構造だ。

しかし、この構造自体が永続しない。
**化石資源は有限だからだ**。

:::highlight
**石油の可採年数：**
確認埋蔵量ベースで約50年。新規発見を含めても100〜150年。
しかし「可採年数」は誤解を招く数字だ。
枯渇は突然来るのではない。**採掘コストが年々上がっていく**。
安い油田から先に掘り尽くされ、残るのは深海、極地、オイルサンド——
コストが高く、環境負荷も大きい。
石油が「なくなる」前に、「高すぎて使えなくなる」。
:::

天然ガスも同じだ。窒素肥料の原料であるアンモニアは天然ガスから作る。
天然ガスの可採年数も約50年。
化学肥料の原料が「高すぎて買えない」時代は、枯渇より先に来る。

:::chain
**化石資源の終わり方：**
安い油田の枯渇 → 採掘コストの上昇
→ ナフサの価格上昇 → プラスチック・肥料の高騰
→ 食料価格の高騰 → 社会不安
→ さらに高コストの油田を開発 → 環境破壊の加速
→ いずれ採掘が経済的に不可能になる
→ **この過程は既に始まっている**
:::

ホルムズ危機のような地政学リスクは、この過程を**一瞬で早送り**する。
50年かけて徐々に上がるはずのコストが、封鎖や空爆で一夜にして跳ね上がる。

化石資源が有限であるという事実は、全ての議論の土台だ。
素材を化石資源に依存する構造は、いずれ必ず終わる。
問題は「終わるかどうか」ではなく、「終わった時に代替があるかどうか」だ。

## 化石資源そのものが、過去の植物と微生物だった

ここで、本章の議論の足元にある最も基本的な事実を確認しておきたい。

**化石資源とは、過去の植物と微生物そのものである**。

:::chain
**化石資源の正体：**
石油 → 数億年前の海洋プランクトン(藻類・動物プランクトン)が堆積し、
        地下で熱と圧力により変成したもの
天然ガス → 同じく海洋・陸上の有機物が分解・変成したもの
石炭 → 約 3 億年前の石炭紀の森林(シダ植物・裸子植物)が堆積・変成
リン鉱石 → 海洋生物の骨や排泄物(リン酸塩)が堆積したもの
カリ鉱石 → 古代の海洋・湖沼の蒸発残留塩(これも生命圏が関わる)
:::

つまり、私たちが「化石資源で作られた物」を使っているとき、
**実は数億年前の植物・微生物が作った物質を使っている**。
プラスチックも、合成繊維も、化学肥料も、起源を辿れば
古代の藻類とシダ植物に行き着く。

:::highlight
**化石資源時代の正体：**
化石資源時代 = **過去の植物・微生物が貯めた素材を取り崩す時代**
バイオ素材時代 = **現在の植物・微生物が作る素材を使う時代**
両者の違いは「時間軸」だけ。**素材の供給者は同じ ── 植物と微生物** である。
:::

これが意味することは大きい。

「化石資源からバイオ素材へ」という転換は、実は
**「過去の生物素材から、現在の生物素材へ」** の転換である。
全く違うものに変えるのではなく、**同じ生物的合成を、現在進行形で
行わせる** ということだ。地球は常にこの仕組みで素材を作ってきた ──
私たちは数億年分の在庫を 100 年で使ったあと、現役の生産ラインに
戻るだけである。

そして、現在の植物と微生物が、過去の植物と微生物と同じ素材を作れる
かどうかは、**現在の植物と微生物の多様性と健康に依存する**。
だからこそ、

- 失われつつある植物の品種を残すこと
- 土壌微生物を回復させること(リジェネラティブ農業)
- 多様な菌類・藻類を育てる場所を確保すること
- 光合成と微生物発酵を活かす技術を磨くこと

これらは、化石資源後の素材インフラの **直接的な準備** である。
別の話ではない。地球の素材生産システムを、過去の貯金から現在の
発電に切り替える、という単一の話だ。

> 化石資源は、地球が数億年かけて貯めた **生物資本の貯金箱** だった。
> 私たちはその貯金を 100 年で使い切りつつある。
> 次に必要なのは、新しい技術ではなく、
> **現在の生物が同じ仕事をできる環境を取り戻すこと** である。

## 化石資源の後に来るもの——微生物と植物

化石資源が枯渇するか、素材の代替が進んで精製が縮小するか——
いずれにせよ、ナフサベースの素材が手に入らなくなる時代が来る。

その時、素材はどこから来るのか。
答えは二つしかない。**微生物と植物**だ。
微生物由来の素材
**バイオプラスチック**——微生物が糖を発酵させてPHA（ポリヒドロキシアルカノエート）を生産。生分解性。
**バイオ繊維**——細菌セルロース。ナタデココと同じ原理。
**バイオ接着剤**——菌糸体ベースの接着剤。
**発酵化学品**——エタノール、乳酸、コハク酸。化学原料として使用可能。
植物由来の素材
**セルロース**——木材、麻、竹から。紙、繊維、建材。
**リグニン**——木材の副産物。接着剤、炭素繊維の原料候補。
**天然ゴム**——ゴムの木から。合成ゴムの代替。
**植物油**——亜麻仁油、桐油。塗料、潤滑剤。
**デンプン**——生分解性プラスチックの原料。
菌糸体（マイセリウム）
キノコの根に当たる菌糸体は、素材革命の鍵を握る。
断熱材、梱包材、建材、皮革代替品として既に実用化が始まっている。
農業廃棄物を基質として成長し、生分解性。
発泡スチロールの直接的な代替候補。
藻類
微細藻類はバイオ燃料だけでなく、バイオプラスチック、
食品添加物、化粧品原料、飼料を生産できる。
CO2を吸収しながら成長し、耕作地を必要としない。
水と光とCO2があれば、どこでも培養可能。

## バイオ素材の正直な限界

ここで正直に言わなければならないことがある。
**バイオ素材への移行は、決して簡単ではない**。
現時点で、バイオ素材は石油化学製品の包括的な代替にはならない。

:::highlight
**バイオ素材の構造的問題：**
**コスト** — バイオプラスチック（PHA）の製造コストは石油由来プラスチックの3〜5倍。大量生産してもコスト差は簡単には縮まらない。
**スケール** — 世界のプラスチック生産量は年間約4億トン。バイオプラスチックは全体の1%未満。100倍のスケールアップは10〜20年では不可能。
**性能** — 耐熱性、強度、耐久性で石油由来に劣る用途が多い。医療器具や半導体のフォトレジストの代替は現時点で存在しない。
**土地** — バイオマス原料の栽培には農地が必要。食料生産と競合する。
**エネルギー** — 発酵・精製プロセスに大量のエネルギーが必要。「脱石油」のためにエネルギーを大量消費するパラドックス。
:::

これは「やがて解決する技術的課題」ではない。
物理的・生物学的な制約だ。
微生物の増殖速度は化学プラントの処理速度に及ばない。
植物の成長速度は需要の成長速度に追いつかない。

### もう一つの致命的な限界 ── バイオ素材は化学肥料に依存している

ここまで挙げた五つの制約に加えて、最も深刻な制約がある。
**バイオ素材の原料は農産物であり、現代の農産物は化学肥料なしでは
大規模に栽培できない**。そして化学肥料は、化石資源そのものから
作られている。

:::chain
**バイオ素材を栽培する化学肥料の化石資源依存：**
窒素肥料(尿素・アンモニア) ← 天然ガス(ハーバー・ボッシュ法)
リン酸肥料 ← リン鉱石(モロッコ・西サハラ70%偏在)＋硫酸＋天然ガス
カリウム肥料 ← カリ鉱石(ロシア・ベラルーシで世界の40%)
農薬(殺虫剤・除草剤・殺菌剤) ← 多くがリン化合物・石油化学
種子処理剤・展着剤 ← 石油化学・界面活性剤
:::

つまり「化石資源を使わずバイオ素材で代替しよう」というシナリオは、
**バイオ素材を栽培するために化石資源由来の化学肥料・農薬を投入する**
という自己矛盾を抱えている。化石資源が細れば、バイオ素材を作るための
農業基盤も同時に細る。

:::highlight
**二重に細る構造：**
化石資源 → ナフサ → 石油化学素材 ── 細る
化石資源 → 化学肥料 → 大規模農業 → バイオ素材原料 ── 同時に細る
:::

しかも、リン酸肥料については 2027 年以降の供給制約が既に見えている
(2026 年 3 月の中国輸出停止、ホルムズ海峡封鎖、ピーク・リン)。
バイオ素材の量産が立ち上がるはずの時間軸より、化学肥料の制約のほうが
先に効いてくる。

詳細は別シリーズ [リン酸肥料が来なくなる日](/phosphorus-and-farming/)
で論じているが、結論だけ言うと:

> **「化石資源の代替としてバイオ素材を量産する」シナリオは、化学
> 肥料に依存しない栽培体系(リジェネラティブ農業・自然農法・菌根菌
> 共生)を同時に確立できる場合にのみ成立する**。

これは「望ましい付加要素」ではなく **必須の前提条件** である。
土壌微生物の回復、菌根菌ネットワークの再構築、PSB(リン溶解菌)の
活用 ── これらが立ち上がらないかぎり、バイオ素材化のシナリオも
立ち上がらない。

### バイオ素材で何でも作れるわけではない

そして、たとえ栽培問題が解決しても、**バイオ素材で代替できない領域は
大量に残る**。次に、自動車・データセンター・核融合炉という、現代文明
の三つの基幹インフラについて、バイオ素材で代替できるかを具体的に
検証する。

### バイオ素材で自動車は作れるか

バイオ素材の限界を最もよく示すのが自動車だ。
EV推進論者は「電気で走ればいい」と言うが、
**車体そのものが化石資源の塊**であることを忘れている。

:::compare
| 部品 | 現在の素材 | バイオ代替 | 現実性 |
| --- | --- | --- | --- |
| ボディパネル | 鋼板・アルミ | — | 金属は化石資源ではない（ただし精錬に大量のエネルギー） |
| バンパー | ポリプロピレン（石油） | バイオプラスチック | 衝撃吸収性能が不足 |
| 内装（ダッシュボード等） | ABS、ポリウレタン（石油） | バイオプラスチック、菌糸体 | 一部可能だが耐熱性に課題 |
| シート | ポリウレタンフォーム（石油） | 天然ラテックス、菌糸体 | コスト数倍。大量生産困難 |
| タイヤ | 合成ゴム（石油）60%＋天然ゴム40% | 天然ゴム100% | ゴムの木の栽培に限界。全量代替は不可能 |
| 塗装 | 石油系溶剤・樹脂 | 植物油系塗料 | 耐候性・光沢で劣る |
| 配線被覆 | PVC、ポリエチレン（石油） | バイオプラスチック | 耐熱性・難燃性が不足 |
| ブレーキホース | 合成ゴム（石油） | 天然ゴム | 耐熱・耐油性が不足。安全に関わる |
| フロントガラス中間膜 | PVB（石油） | 代替なし | 安全ガラスに不可欠 |
:::

EVであっても、バッテリー以外の車体構造は内燃機関車と大差ない。
同じ量の石油由来素材を使う。
**EVは「走る時」に石油を使わないだけで、「作る時」には石油なしに作れない。**

:::highlight
**自動車が突きつける現実：**
バイオ素材で自動車の全部品を代替することは、現時点で不可能だ。
安全に関わる部品（ブレーキホース、タイヤ、安全ガラス中間膜）は特に代替が難しい。
つまり、自動車を作り続ける限り、石油由来素材が必要であり、
石油精製は続き、ガソリンも軽油も出てくる。
これが「全車EV化は無謀」のもう一つの根拠だ。
:::

### バイオ素材でデータセンターは作れるか

「脱炭素」の旗を振るIT企業自身が、化石資源の塊の上で商売している。

:::compare
| 部品 | 現在の素材 | バイオ代替 | 現実性 |
| --- | --- | --- | --- |
| サーバー筐体 | ABS、ポリカーボネート（石油） | バイオプラスチック | 耐熱性・難燃性が不足 |
| 基板（PCB） | エポキシ樹脂＋ガラス繊維（石油） | 代替なし | 半導体実装に必要な精度・耐熱性 |
| 半導体パッケージング | エポキシモールド（石油） | 代替なし | ナノスケールの精度が必要 |
| フォトレジスト | 感光性樹脂（石油化学） | 代替なし | 半導体製造の根幹 |
| 冷却液ホース | 合成ゴム、PVC（石油） | 天然ゴム | 耐薬品性が不足 |
| ケーブル被覆 | PVC、ポリエチレン（石油） | バイオプラスチック | 難燃規格を満たせない |
| 光ファイバー被覆 | アクリレート（石油） | 代替なし | 光学特性の精度が必要 |
| 床材（静電防止） | 石油系ビニル | 代替なし | 静電気放電は機器破壊に直結 |
:::

AIも、クラウドも、SNSも、全て石油化学素材で作られた箱の中で動いている。
半導体の製造にはフォトレジストが不可欠で、これは石油化学でしか作れない。
**デジタル文明は石油文明の上に乗っている。**

### バイオ素材で核融合炉は作れるか

核融合は「究極のクリーンエネルギー」と呼ばれる。
しかし、その炉を**何で作るのか**を考えた人間は少ない。

:::compare
| 部品 | 現在の素材 | バイオ代替 | 現実性 |
| --- | --- | --- | --- |
| プラズマ対向壁 | タングステン、ベリリウム | — | 金属。代替の議論以前 |
| ブランケット構造材 | 低放射化フェライト鋼 | — | 金属 |
| 超伝導コイル絶縁 | エポキシ樹脂＋ガラス繊維（石油） | 代替なし | 極低温（-269°C）で機能する絶縁材 |
| 真空容器のシール | フッ素ゴム、合成ゴム（石油） | 代替なし | 超高真空を維持。天然ゴムでは不可能 |
| トリチウム配管シール | 特殊フッ素樹脂（石油） | 代替なし | 放射性トリチウムの漏洩防止 |
| 診断機器の光学窓 | 合成石英＋石油系コーティング | 代替なし | 高放射線環境での光学計測 |
| 制御系ケーブル | 耐放射線ポリイミド（石油） | 代替なし | 通常のプラスチックは放射線で劣化 |
:::

:::highlight
**核融合のパラドックス：**
核融合は「石油に代わるエネルギー」として期待されている。
しかし、核融合炉自体の建設に石油化学素材が不可欠だ。
超伝導コイルの絶縁材、真空シール、耐放射線ケーブル——
これらは全て石油化学でしか作れない。
**石油なしには、石油に代わるエネルギーの炉を作ることすらできない。**
:::

自動車、データセンター、核融合炉——
現代文明のあらゆるインフラが石油化学素材なしには成り立たない。
バイオ素材はこのうちの一部を代替できるに過ぎない。
これが、化石資源を大切に使わなければならない最も根本的な理由だ。

## それでもやるしかない

しかし、化石資源は有限だ。
石油は掘れば減る。天然ガスも掘れば減る。
100年後、200年後、必ず枯渇する。
ホルムズ危機のような地政学リスクで、突然止まることもある。

バイオ素材が石油化学に劣ることは事実だ。
しかし、石油化学がいつまでも使える前提で暮らすことはできない。
「劣るからやらない」のではなく、
「劣るけど今から始めなければ間に合わない」。

:::chain
**移行の時間軸：**
バイオ素材の技術開発 → 10〜20年
製造スケールの拡大 → 20〜30年
社会インフラの転換 → 30〜50年
石油が確実に使える残り時間 → **不明**（地政学リスクで突然止まりうる）
:::

だからこそ、今ある化石資源は大切に使わなければならない。

## 将来の自動車をどうするか ── 今、決めなくていい

ここまで本章では「全車 EV 化を急ぐと精製構造が破綻する」「ハイブリッド
車は現状の精製副産物を使い切る合理解」「バイオ素材で自動車を全部
代替するのは現時点で不可能」という **2026 年時点の構造分析** を示してきた。

しかし、20 年・30 年・50 年後の自動車がどうあるべきかは、
**今、答えを出す問題ではない**。

未来は、今から見えている要素が複雑に絡み合って決まる。

:::highlight
**20 年スパンで動く要素：**
バッテリー技術の進歩(全固体電池、ナトリウムイオン、半固体)
自動運転と所有モデルの変化(車を所有しない人の増加)
都市集中の解消(別章) → そもそも長距離移動が必要な人口が減る
リモートワークと土地ベース労働への移行
公共交通の再設計
バイオ素材の段階的進歩
燃料合成技術(eFuel、合成航空燃料)
化石資源の制約タイミング(地政学リスクで前倒しの可能性)
リン酸肥料制約による農業構造の変化
:::

これらが 20 年後にどう組み合わさるかは、**今、予測できない**。
20 年前(2006 年)に「2026 年の自動車はどうあるべきか」を予測した
人で、現在の状況を当てた人はいない。スマートフォン以前、自動運転
以前、リチウムイオン電池の量産普及以前である。

だから、「未来の自動車像」を今、決めようとすればするほど、現実から
離れる。

:::highlight
**今の私たちにできること:**
1. 現状の構造を正確に分析する(全車 EV 化を急ぐと精製副産物の
   行き場がなくなる、など)
2. **ロックインを避ける** ── 「これが将来の答えだ」を法律・補助金で
   固定しない
3. 変化に対応できる社会のインフラを準備する
4. 現実の課題になった時に、その時点の状況で真剣に考える
:::

> 「将来の自動車をどうするか」を 2026 年の知見で決めようとすれば、
> 必ず外す。決めるべきではない。
> 決めるべきは、**変化が来た時に対応できる構造を作っておくこと** だ。

## 変化に対応できる社会を作る

ここまで本章で示してきたのは、化石資源と現代文明の構造的相互依存
であり、その制約だ。しかし結論は「だから、今、すべての答えを決めよう」
ではない。**変化に対応できる社会を作ろう** である。

### 適応性の根幹は「依存の分散」と「多様性」

そのために最も重要なのが、**特定のものに依存しすぎないこと、
多様性を重視すること** である。

特定の何かに依存すればするほど、その何かが変化した時に脆弱になる。
裏返せば、**依存を分散させ、多様性を保てば、何が変わっても対応できる**。
これは生態系から学んだ最も基本的な原理だ。モノカルチャーは脆弱、
多種混植は強靭。社会も同じ構造で動く。

:::compare
| 依存しすぎてはいけない | 多様性で代替する |
| --- | --- |
| 特定の化石資源(石油・天然ガス・リン鉱石) | バイオ素材+再エネ+蓄積資源の組み合わせ |
| 特定の輸入国(モロッコ・西サハラのリン、ロシアのカリ) | 国産化・代替源・リサイクル・自然農法 |
| 特定の作物・品種(F1・モノカルチャー) | 多品目混植・在来種・自家採種 |
| 特定の企業・サプライチェーン(SaaS・SIer・特定 OS) | OSS・自前構築・代替ベンダー |
| 特定のエネルギー源(電力・ガソリン・軽油) | 多様なエネルギー源と用途別の最適化 |
| 特定の地域(東京一極) | 地方分散・分散型インフラ |
| 特定の通貨(円・ドル) | 多通貨保有・実物資産 |
| 特定のテクノロジー(プロプライエタリ AI) | OSS AI・ローカル推論 |
| 特定の企業の特定製品(Office・Photoshop・Salesforce) | 互換性のある複数選択肢 |
| 特定の答え・正解像 | 答えを変えられる柔軟性 |
:::

> 「これだけあれば大丈夫」を作らないこと。
> 「これがないと駄目」を作らないこと。
> ── これが多様性の本質である。

### 素材としての植物多様性を残す

多様性のなかでも、特に重要なのが **植物の多様性** だ。
食料の話だけではない。**素材と資源としての植物多様性** が、化石資源後
の社会の基盤になる。

化石資源以前、人類はあらゆる素材を植物から取り出していた。

:::compare
| 用途 | 利用されてきた植物の例 |
| --- | --- |
| 繊維 | 麻・苧麻(ラミー)・棉・芭蕉・葛・藤・楮(こうぞ)・三椏(みつまた)・雁皮(がんぴ) |
| 建材・木工 | 杉・檜・松・楢・栗・桐・竹・欅・桑・檀(まゆみ) |
| 接着剤・塗料 | 漆・松脂・柿渋・桐油・亜麻仁油・荏胡麻油 |
| ゴム・樹脂 | パラゴムノキ・グアユール・ロシアタンポポ・ヤトロファ |
| 染料 | 藍・紅花・茜・紫根・刈安・蘇芳 |
| 医薬品原料 | キニーネ・モルヒネ・タキソール(イチイ)・薬用植物多数 |
| 香料・精油 | 檜・薄荷・薔薇・各種ハーブ・柑橘 |
| 油脂 | 菜種・大豆・椰子・パーム・荏胡麻・亜麻仁・桐 |
| 紙 | 楮・三椏・雁皮・木材パルプ各種 |
| 食品包装・容器 | 朴葉・笹・竹皮・椰子の葉・蓮葉 |
| バイオプラスチック原料 | 馬鈴薯・トウモロコシ・サトウキビ・キャッサバ |
| 菌糸体培養基質 | 各種農業残渣(藁・籾殻・木屑) |
:::

しかし、化石資源によるプラスチック・合成繊維・合成染料・合成医薬品
の普及で、これら **植物素材の利用ノウハウと品種多様性** が急速に
失われている。

- 国産漆の生産量は 100 年で 1/100 以下に
- 在来綿の生産はほぼ消滅(明治期に米綿に切り替え)
- 染料植物の栽培は工芸用に細々と残るのみ
- 薬用植物の生産は中国輸入依存
- 桐・檀などの建材種の植林は激減
- 苧麻・楮・三椏・雁皮の生産農家は高齢化で消滅寸前

化石資源後の世界で再びこれらが必要になった時、
**「植物自体が残っていない」「育て方が分からない」「加工技術が
失われた」** という事態を避けなければならない。一度途絶えた品種は
復元が極めて困難であり、加工技術は文書化しても身体知の部分は失われる。

:::highlight
**残すべき三層：**
1. **遺伝資源としての種・品種** ── 種子バンク、農家の自家採種、
   植物園、圃場、樹木の現地保存
2. **栽培・育成のノウハウ** ── 地域の伝統知、農家・職人の技術伝承、
   気候風土に合った品種選定の経験
3. **素材化の技術** ── 漆掻き、木挽き、紙漉き、藍染、薬草加工、
   油搾り ── これらは AI で文書化はできるが、**身体技能は人間に
   しか残せない**
:::

化石資源後の素材供給の鍵は、**多様な植物の種(たね)と種(しゅ)**
にある。これは「保護すべき文化遺産」ではない。未来の素材インフラ
そのものである。

> 化石資源時代に失われかけている植物の多様性と素材化技術は、
> ポスト化石時代の **物理的な生存基盤** である。
> 経済合理性で残すのではない。**残さなければ、後で作れない** から残す。

### 適応性のある社会の条件

依存の分散と多様性を、社会システムのレベルで具体化すると、こう
なる。

:::chain
**適応性のある社会の条件：**
化石資源の素材機能を理解している → 不意打ちにならない
ロックインを避ける政策設計 → 「正解」を法で固定しない
リジェネラティブ農業の実装 → 化学肥料が細っても食料は維持できる
分散型インフラ → 中央集約の脆弱性を回避
ローカルな生産・流通の再構築 → 物流ショックへの耐性
教育と AI による学習速度の向上 → 状況変化への素早い応答
個人と小組織の自律性 → 大企業・大国の判断ミスから切り離されている
:::

「いつ・どんな順序で変化が来るか」は予測できない。
気候変動が先か、地政学リスクが先か、技術ブレークスルーが先か。
2027 年のリン酸肥料制約は確実に近い。しかしその他は分からない。

しかし、**「変化が来た時に、どれだけ早く・賢く対応できるか」は、
今から準備できる**。これが本シリーズの構造分析が指し示す方向である。

> 未来を予測するのではなく、**未来に対応する力を高める**。
> 答えを今、決めるのではなく、**答えを変えられる社会を残す**。
> 一つに賭けるのではなく、**多様性を保つ**。

## 化石資源は大切に使え

化石資源が有限であることを理解すれば、
今の使い方がいかに愚かかが分かる。

:::highlight
**現在の使い方：**
石油の86%をエネルギーとして**燃やしている**。
燃やせばCO2になり、二度と素材には戻らない。
残り14%が化学原料（ナフサなど）として素材に使われている。
86%を燃やす余裕は、本来ないはずだ。
:::

しかし、ここに構造的な矛盾がある。

:::highlight
**精製の構造的制約：**
原油は精製すると、ナフサ・ガソリン・軽油・重油が**同時に**出てくる。
ナフサだけを取り出すことはできない。
現在の量の化学原料（ナフサ）が必要な限り、原油の精製は続く。
精製すれば、燃料も必ず出てくる。
つまり、「石油を燃やすのをやめろ」は、精製の構造上、単純には成り立たない。
:::

ではどうするか。

**短期的には、精製で出てくる燃料を使いながら、需要の構造を変える。**
使い捨てプラスチックの浪費、過剰包装、短寿命の製品設計——
これらを減らせば、必要なナフサの量が減り、精製量が減り、燃料の副産物も減る。

**中長期的には、バイオ素材でナフサ需要そのものを減らす。**
ナフサ需要が減れば、精製量が減り、燃料の副産物も減る。
そこに再生可能エネルギーの拡大を組み合わせれば、
石油精製を段階的に縮小できる。

再生可能エネルギーへの転換は、「環境のため」だけではない。
**有限な化石資源を素材として長く使うため**でもある。
しかし、素材需要がある限り精製は止められない——
この構造的制約を無視した「脱石油」は空論だ。

:::compare
| 戦略 | 短期（5年） | 中期（20年） | 長期（50年） |
| --- | --- | --- | --- |
| エネルギー | 再エネ拡大。燃料副産物の依存を減らす | 再エネ主力化 | 精製量の大幅縮小 |
| 素材 | 使い捨て削減。ナフサ需要を減らす | バイオ素材のスケールアップ | バイオ素材が主力 |
| 土壌 | 再生農業の開始 | 土壌微生物の回復 | 生物学的インフラの確立 |
| 備蓄 | ナフサ・肥料の戦略備蓄を新設 | 備蓄量の拡大 | 備蓄不要（自給体制） |
:::

## 福岡正信が見ていた未来

福岡正信は1970年代に「わら一本の革命」を書いた。
石油危機の直後だ。
しかし、彼が批判したのはエネルギー依存ではなく、
化学肥料と農薬への依存——つまり、石油の素材機能への依存だった。

:::quote
自然農法は、石油文明が終わった後の農業モデルではない。
石油文明が始まる前から存在していた農業モデルだ。
微生物と植物が素材を供給し、人間はその中で暮らしていた。
それが本来の姿であり、戻るべき場所だ。
:::

リジェネラティブ農業は、単に「化学肥料を使わない」農業ではない。
土壌微生物を育て、菌根菌のネットワークを再構築し、
植物と微生物の共生系を回復させる——
これは、化石資源後の世界で素材を供給するシステムの基盤でもある。

## 構造的に見える未来

化石資源後の世界は、「不便になる」世界ではない。
素材の供給源が変わる世界だ。

:::chain
**素材供給の転換：**
石油精製 → ナフサ → プラスチック → 大量生産・大量廃棄
微生物発酵 → バイオポリマー → 生分解性素材
植物栽培 → セルロース・リグニン → 繊維・建材
菌糸体培養 → マイセリウム → 梱包材・断熱材
藻類培養 → バイオオイル → 化学原料
:::

この転換には、膨大な量の「生物学的インフラ」が必要だ。
微生物の培養施設、植物の栽培システム、菌糸体の生産拠点——
これらは全て、土壌の健康と生物多様性の上に成り立つ。

そして決定的に重要なのは、**この生物学的インフラは、化学肥料に依存
しない栽培体系の上にしか立たない** ということだ。化学肥料に依存した
ままバイオ素材を量産しようとすれば、化石資源の制約が先に効いて、
量産自体が成り立たなくなる。

つまり、化石資源後の世界の準備は、
**土壌の再生** ── リジェネラティブ農業の確立 ── から始まる。
これは「環境のため」でも「農業の話」でもない。
**現代文明全体のインフラを再構築するための、必須の前提条件** である。

:::highlight
**結論：**
現代文明の便利さの正体は、化石資源から作られた素材だ。
エネルギーは代替できるが、素材の代替には微生物と植物が必要になる。
しかしバイオ素材も化学肥料がないと栽培できない。
化学肥料の原料も化石資源から来る。
バイオ素材で何でも作れるわけでもない(自動車・半導体・核融合炉)。
化石資源後の世界に備える唯一の方法は、
微生物と植物を育てる力を取り戻すこと、
そして **化学肥料に依存しない栽培体系を同時に確立すること** ──
つまり、リン酸肥料の供給制約が顕在化する前に、土壌を再生することだ。
:::
