## 三つの崩壊が同時に来る

日本はいま、三つの構造的崩壊に同時に直面している。

:::chain
1. 医療 → 透析だけで年間1.6兆円。化石資源に完全依存
2. 年金 → 現役世代の社会保険料が30%を超えた
3. 食料 → 化学肥料に依存。サプライチェーンは中東に繋がっている
:::

これらは別々の問題ではない。
構造的に見れば、全て同じ根から生えている。
**自然から離れたことの代償だ。**

## 透析医療——化石資源で命を繋ぐ構造

日本の透析患者数は2024年末で337,414人。
国民の約367人に1人が透析を受けている。
患者一人あたり年間約480万円。マクロ総額は**年間約1.6兆円**。

:::chain
**透析医療の化石資源依存：**

ダイアライザー（人工腎臓） → ポリスルホン膜＋ポリカーボネート外筒 → **石油化学製品**
血液回路・穿刺針 → ポリ塩化ビニル（PVC）チューブ → **石油化学製品**
透析液バッグ → プラスチック素材 → **石油化学製品**
超純水の精製 → RO装置の常時稼働 → **大量の電力**
透析液の加温 → 37度まで加熱 → **大量の電力**
14万台のコンソール → 空調・駆動 → **大量の電力**
:::

透析治療の63.3%が血液透析濾過（HDF）——従来型より大量の超純水と電力を消費する。
在宅透析はわずか3.4%。世界最低水準だ。
全国4,512施設に148,339台のコンソール。
この巨大インフラは、化石資源が止まれば機能しない。

:::highlight
**2026年イラン戦争の直撃：**
オイルショックにより電力・材料・物流費が同時に高騰。
透析施設のコスト構造が崩壊する。
患者一人あたりの年間費用は480万円から600万円超へ。
患者数は減少しているのに（337,414人→推計325,380人）、
総医療費は1.6兆円から**2兆円規模へ上方爆発**する。
人口動態の恩恵が、インフレに完全に食われる。
:::

2026年度の診療報酬改定で、光熱水費の補填はわずか**+0.09%**。
オイルショックの現実とは絶望的に乖離している。
中小の透析クリニックは経営不能に陥り、
「透析難民」が発生するリスクが現実化する。

## 社会保険料30%——現役世代の限界

透析は氷山の一角だ。
問題は、日本の社会保障制度全体の構造にある。

:::chain
**現役世代が背負う負担：**

健康保険料 → 約10%
厚生年金保険料 → 約18.3%
介護保険料 → 約1.8%
雇用保険料 → 約0.9%

**合計 → 約31%**（労使折半前の実質負担）
:::

給与の3割以上が、働く前に消える。
しかも、この比率は毎年上がり続けている。
少子高齢化が進めば、40%、50%になる計算だ。

:::highlight
**構造的に成立しない数学：**
1970年：現役世代8.5人で高齢者1人を支えていた
2020年：現役世代2.1人で高齢者1人を支えている
2040年：現役世代1.5人で高齢者1人を支える予測
1.5人で1人を支える社会保障は、どんな制度設計でも破綻する。
:::

## 高齢者が年金で遊ぶ社会は成立しない

さらに根本的な問題がある。

現在の社会モデルは「65歳まで働く→年金で余生を過ごす」だ。
年金制度の前提は「余生は短い」だった。
1960年代、日本人の平均寿命は65〜67歳。年金支給は55歳から。
余生は10年程度の想定だった。

いま、平均寿命は84歳。年金支給開始は65歳。
余生は20年。しかも、その20年間に医療費が膨大にかかる。

:::chain
**年金の構造的矛盾：**

長寿化 → 年金支給期間が20年以上に
少子化 → 支える現役世代が減少
医療の高度化 → 高齢者の医療費が増大
社会保険料 → 現役世代の30%超

働く人が減り、遊ぶ人が増える → **数学的に破綻**
:::

「高齢者が年金で遊ぶ」モデルは、人口が増え続ける時代にしか成立しなかった。
その時代は終わった。

:::highlight
**逆転の発想が必要だ：**
現行モデル：若者が働く → 税金と保険料を払う → 高齢者が医療と年金を受ける
逆転モデル：全世代が「生きる力」を持つ → 医療に頼らない → 年金に頼らない
「病気を治す」のではなく「病気にならない」。
「年金で生活する」のではなく「年金が不要な暮らし」。
これは福岡正信の「何もしない農業」と同じ構造の逆転だ。
:::

人間は楽をしたい。しかし、楽をした結果がこの崩壊だ。
「多くを持つこと」は「楽に生きること」ではなかった。
自然から離れるほど、生きるコストは上がる。

## アメリカ農業の構造的破綻

「日本の農業は非効率だ、アメリカのように大規模化すべきだ」
——この主張は、アメリカ農業の構造を全く理解していない。

:::chain
**アメリカ農業の実態：**

大規模モノカルチャー → 同じ作物を広大な土地に → 土壌微生物の多様性が消滅
化学肥料への完全依存 → 窒素・リン・カリウムを外部から投入 → 土壌は「培地」に退化
大量の農薬 → 除草剤（グリホサート等）→ 土壌微生物をさらに殺す
大型機械 → 土壌を圧縮 → 水と空気の循環を破壊
灌漑への依存 → オガララ帯水層の枯渇 → 回復に数千年
:::

:::highlight
**表土の喪失——取り返しのつかない損失：**
アメリカの農地は、過去150年で表土の約50%を失った。
表土1インチ（2.5cm）の形成には500〜1,000年かかる。
大規模農業は、数千年かけて蓄積された土壌資本を、
数十年で使い果たしている。
これは採掘であって、農業ではない。
:::

:::highlight
**オガララ帯水層——枯渇する地下水：**
アメリカ中西部の穀倉地帯を支える巨大地下水脈。
年間の汲み上げ量が、自然補給量の数倍〜数十倍。
カンザス州やテキサス州では、すでに水位が30m以上低下。
完全な回復には数千年〜数万年かかる。
アメリカの「食料安全保障」は、枯渇する地下水の上に建っている。
:::

:::compare
| 項目 | アメリカ大規模農業 | 自然農法 |
| --- | --- | --- |
| 土壌 | 消耗品（採掘して使う） | 生きたシステム（育てる） |
| 肥料 | 外部依存（化石資源由来） | 不要（微生物が供給） |
| 水 | 地下水を採掘（枯渇する） | 雨水＋土壌の保水力 |
| 農薬 | 必須（モノカルチャーの宿命） | 不要（生態系バランス） |
| エネルギー | 大量消費（機械＋輸送） | 最小限（人力＋自然力） |
| 期間 | 持続不可能（数十年で土壌崩壊） | 持続可能（数千年の実績） |
| 危機耐性 | 極めて脆弱（サプライチェーン依存） | レジリエント（自立型） |
:::

アメリカ農業は「効率的」に見えるが、
それは土壌と地下水という資本を食い潰しているからだ。
銀行口座の残高を使い果たしながら「収入が多い」と言っているに等しい。

## 日本の本当の資源——水

「日本には資源がない」

これは日本で最も広く信じられている誤解だ。

日本には**水**がある。これが最大の天然資源だ。

:::chain
**水が豊かな国の構造的強み：**

年間降水量 → 日本は約1,700mm（世界平均の約2倍）
水源 → 山岳地帯が国土の73%。天然の貯水・浄水システム
河川密度 → 極めて高い。どこでも水にアクセスできる
地下水 → 豊富で清浄
温暖湿潤気候 → 植物の成長に最適
:::

:::highlight
**水がなぜ最大の資源か：**
石油は代替できる（太陽光、風力、核融合）。
水は代替できない。
農業には水が必要。
微生物には水が必要。
植物には水が必要。
人間には水が必要。
アメリカのオガララ帯水層は枯渇している。
中東は慢性的な水不足。
中国北部は深刻な水危機。
日本は、毎年空から1,700mmの水が降ってくる。
これを「資源がない」と言うのは、
石油とレアメタルしか「資源」と認識していないからだ。
:::

水があれば、植物が育つ。
植物が育てば、微生物が育つ。
微生物が育てば、土壌が再生する。
土壌が再生すれば、食料が生まれる。
食料があれば、人間は生きられる。

:::chain
**日本の本当の資源循環：**

豊富な雨水 → 多様な植物 → 土壌微生物 → 肥沃な土壌 → 食料
↑ ← ← ← ← ← ← ← ← ← ← ← ← ← ← ← ←（循環）← ↓
:::

石油は掘れば減る。水は循環する。
これが「資源」の本質的な違いだ。

## 多様な植物と暮らしの知恵

日本列島は南北3,000kmにわたり、
亜熱帯から亜寒帯まで、驚くべき植物の多様性を持つ。

:::highlight
**失われつつある知恵：**
どの野草が食べられるか。
どの植物が薬になるか。
どの木が建材になり、どの草が繊維になるか。
いつ種を蒔き、いつ収穫するか。
どの土地にどの作物が合うか。
この知恵は、数千年かけて蓄積されてきた。
しかし、スーパーマーケットとコンビニの時代に、
急速に失われている。
:::

この知恵を残さなければならない。

化石資源が止まり、サプライチェーンが崩壊し、
スーパーの棚が空になった時、
生き残るのは「自分の土地で食べ物を作れる人」だ。
そのために必要なのは、最先端テクノロジーではない。
水と、植物と、暮らしの知恵だ。

## 福岡正信の思想——生きるためにどうするか

全ての構造分析は、一つの問いに収束する。

**生きるためにどうするか。**

福岡正信の自然農法は、この問いへの答えだ。

:::chain
**福岡正信の四つの原則の本質：**

無耕起 → 土壌の構造を壊さない → 微生物が生きる
無肥料 → 外部依存をゼロにする → サプライチェーンに縛られない
無農薬 → 生態系を殺さない → 自然の防御力を活かす
無除草 → 光合成面積を最大化 → Light Farmingの原理
:::

そしてChristine Jones博士のLight Farmingが、
この思想の科学的根拠を与えた。

:::chain
**Light Farming——福岡正信の科学的証明：**

太陽光 → 光合成 → 植物が糖を作る
糖 → 根から土壌へ滲出（Liquid Carbon）
Liquid Carbon → 土壌微生物のエネルギー源
土壌微生物 → 植物に必要なミネラルを可溶化して供給
植物 → さらに光合成 →（正のフィードバックループ）

**雑草も光合成する。草を残せば、Liquid Carbonの供給源が増える。**
福岡正信の「無除草」は、Light Farmingの最大化戦略だった。
:::
