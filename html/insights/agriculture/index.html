Structural Analysis 01

農業の間違い

自然農法は理想論ではない。サバイバル戦略である。

2025.04.03

構造分析シリーズ "01"

製油所一つで崩壊する農業モデル

現代農業は化学肥料に依存している。この事実は誰もが知っている。 しかし、その化学肥料がどこから来るのかを追跡した人間はほとんどいない。

リン酸肥料を作るには硫酸が必要だ。 硫酸の原料である硫黄は、石油精製の副産物として回収される。 窒素肥料（尿素）は天然ガスから合成される。 食品包装材の原料であるナフサも、石油精製から生まれる。

化学肥料の生産ルート：
石油精製 → 硫黄（副産物）
→ 硫酸
→ リン鉱石の処理
→ リン酸肥料
天然ガス → 水素 + CO2
→ アンモニア
→ 尿素（窒素肥料）
石油精製 → ナフサ
→ プラスチック
→ 食品包装材

つまり、石油精製施設を物理的に破壊すれば、三つの危機が同時に発生する。 肥料が作れない。食品を包装できない。物流が止まる。

「封鎖」と「破壊」は根本的に異なる。 封鎖は解除すれば翌日から供給が再開する。 破壊された製油所は修復に3〜5年かかる。 現代農業の食料生産能力は、この3〜5年の間、大幅に低下する。

単一障害点（Single Point of Failure）： 化学肥料依存の農業モデルは、石油精製という単一障害点を持つ。 ITの世界では、単一障害点を持つシステムは設計ミスとされる。 農業では、この設計ミスが100年以上放置されてきた。

アメリカの原油では代替できない

「湾岸の油田が止まっても、アメリカはシェールオイルがあるから大丈夫だ」—— これは構造を理解していない反論だ。

原油には種類がある。アメリカのシェールオイルは 「ライトスイート原油（light sweet crude）」と呼ばれる。 「スイート」とは硫黄含有量が少ないという意味だ。 一方、中東・湾岸の原油は「サワー原油（sour crude）」——硫黄含有量が高い。

原油の種類と硫黄：
米国シェールオイル（ライトスイート）→ 硫黄含有量 0.1〜0.5%
中東サワー原油（アラビアンヘビー等）→ 硫黄含有量 2〜3%
硫黄回収量の差 → 4〜30倍

世界の硫黄生産の大部分は、サワー原油の精製時に脱硫プロセスで回収される副産物だ。 アメリカがいくらシェールオイルを増産しても、 硫黄の供給量は湾岸の精製能力を代替できない。

つまり、アメリカの「エネルギー独立」は「素材独立」を意味しない。 ガソリンや軽油は自国で作れても、 リン酸肥料の原料となる硫黄は湾岸のサワー原油精製に依存している。

ヘリウムも同じ構造

硫黄だけではない。ヘリウムも同じ構造的問題を抱えている。

ヘリウムは天然ガス田から回収される副産物だが、 すべてのガス田からヘリウムが出るわけではない。 地質学的条件が揃った特定のガス田——米国テキサス、カタール、アルジェリアなど—— からしか回収できない。

ヘリウムはMRI、半導体製造、ロケット燃料の冷却、光ファイバー製造に不可欠だ。 代替手段はない。合成もできない（元素だから）。 リサイクルも困難（軽すぎて大気圏外に逃げる）。

「湾岸がなくても大丈夫」の嘘： 米国の原油は硫黄が少ない → リン酸肥料の硫黄を代替できない 特定のガス田だけがヘリウムを産出する → 医療・半導体が停止するリスク 「エネルギー独立」は「素材独立」ではない。 エネルギーの問題と素材の問題を混同している限り、本当のリスクは見えない。

土壌微生物は1年で回復する

化学肥料を投入し続けた土壌では、土壌微生物の多様性が著しく低下する。 菌根菌のネットワークが分断され、植物は自力で養分を吸収する能力を失う。 化学肥料がなければ作物が育たない——これは事実だが、永続的な事実ではない。

Christine Jones博士の研究が示すのは、化学肥料の投入を止め、 被覆作物と多様な植物を導入すれば、 土壌微生物群集は驚くほど速く回復するということだ。 多くの場合、1〜3年で菌根菌のネットワークが再構築され始める。

土壌回復のプロセス：
化学肥料停止 → 被覆作物の導入
→ 根からの滲出液が微生物を養う
→ 菌根菌ネットワーク再構築
→ 植物-微生物の共生回復
→ 養分の自然供給
→ 化学肥料が不要になる

問題は「土壌が回復できるかどうか」ではない。 回復できることは科学的に証明されている。 問題は「回復を始める前に危機が来る」ことだ。

30年間の気候変動議論は順序が逆だった

1990年代から30年間、気候変動の議論はCO2排出量の削減に集中してきた。 再生可能エネルギー、電気自動車、カーボンプライシング—— すべて「排出を減らす」ことを目的としている。

しかし構造的に見れば、順序が逆だった。

気候変動議論の順序の誤り： 議論されてきた順序：CO2排出削減 → 再生可能エネルギー → 気温上昇の抑制 構造的に正しい順序：土壌の再生 → 生物多様性の回復 → 炭素固定 → 気候の安定化

土壌は大気と海洋を合わせたよりも多くの炭素を貯蔵している。 過去150年間で、農業土壌は炭素の30〜75%を失った。 この炭素が大気中に放出されたことが、気候変動の主要な要因の一つだ。

Christine Jones博士のLight Farmingが示すように、 生きた根からの液体炭素は、地上バイオマスの約5倍の効率で土壌に安定的に固定される。 つまり、植物を育て、根を張らせ、土壌微生物と共生させることが、 最も効率的な炭素固定の方法だった。

生物多様性の回復と土壌の再生が先。エネルギー転換はその後でいい。 30年間、この順序が逆だった。

自然農法はサバイバル戦略である

福岡正信の四原則——不耕起、無肥料、無農薬、無除草——は、 しばしば「理想論」や「哲学」として語られる。 実用的ではない、規模が拡大できない、収量が低い、と批判される。

しかし、前述の構造分析を踏まえれば、 自然農法の本質は全く異なるものとして見える。

自然農法の構造的意味：
無肥料 = 石油精製への依存ゼロ（単一障害点の排除）
不耕起 = 菌根菌ネットワークの維持（養分供給の自律性）
無農薬 = 化学サプライチェーンへの依存ゼロ
無除草 = 光合成面積の最大化（Light Farmingの原則そのもの）

無除草——福岡正信の独自性とLight Farmingへの接続

不耕起、無肥料、無農薬は、有機農業やパーマカルチャーとも共通する原則だ。 しかし、無除草は福岡正信に固有の原則であり、 他の自然農法や有機農業とは根本的に異なる。

有機農業でも雑草は敵だ。手で抜く。マルチで覆う。 パーマカルチャーでも、デザインによって雑草を管理する。 福岡正信だけが、「草を抜くな」と言った。

なぜか。 Christine Jones博士のLight Farmingが、50年後にその理由を科学的に証明した。

無除草 → Light Farmingの因果連鎖：
草を抜かない → 土壌が常に緑で覆われる
→ 光合成面積が最大化される
→ 光合成産物（糖）が根から滲出される
→ 土壌微生物が養われる
→ 菌根菌ネットワークが拡大する
→ 液体炭素が土壌に安定的に固定される
→ 土壌の炭素貯蔵量が増加する
→ 土壌構造が改善し、保水力が向上する

Jones博士のLight Farmingの第一原則は 「緑は良い、一年中緑はさらに良い（Green is good. Year-long green is even better.）」だ。 裸地は光合成能力ゼロ。裸地は炭素源であり、浸食にも脆弱。 土が見えていれば、炭素と窒素を失っている。

福岡正信の「草を抜くな」は、まさにこの原則を実践していた。 「雑草」と呼ばれる植物は、土壌を緑で覆い、光合成を続け、 根から液体炭素を土壌に送り続ける。 除草は、この光合成→液体炭素→土壌微生物の循環を断ち切る行為だ。

福岡正信の無除草 = Jones博士のLight Farming： 福岡正信は哲学として「草を抜くな」と言った。 Christine Jones博士は科学として「裸地を作るな」と言った。 50年の時間差があるが、同じ構造を見ている。 多様な植物が常に土壌を覆い、光合成し、根から液体炭素を送り続けること—— これが土壌再生の核心であり、他の自然農法にはない福岡正信の独自性だ。

自然農法は、化学肥料サプライチェーンに対する依存を そもそも持たない農業モデルだ。 製油所が破壊されても、天然ガスの供給が止まっても、 自然農法の畑は影響を受けない。

しかもそれだけではない。 無除草によって光合成面積を最大化し、 液体炭素を通じて土壌に炭素を固定し続ける—— これは単なる「依存ゼロ」ではなく、 積極的に土壌を再生し続けるシステムだ。

自然農法は「何もしない」農法ではない。
外部システムの崩壊に対して「何も失わない」農法だ。

正直な課題——野菜が雑草に負ける

ここまで構造的な正しさを示してきた。 しかし、正直に言わなければならないことがある。 無除草での栽培は、実践としては極めて難しい。

理論は明快だ。草を抜かなければ光合成面積が最大化され、 液体炭素が土壌微生物を養い、土壌が再生される。 しかし現実の畑では、こうなる。

最大の課題——野菜が雑草に負ける：
雑草は強い。野菜は弱い。これが現実。
雑草 → 何千年もその土地に適応してきた。成長が速い。根が深い。
野菜 → 人間が品種改良で作った。肥料と除草を前提に設計されている。
草を抜かない → 雑草が光を奪う → 野菜が育たない

これは「移行期間」の問題ではない。 野菜という植物が、そもそも雑草との競争に弱いという 構造的な問題だ。 現代の野菜品種は、化学肥料と除草を前提とした環境で 選抜されてきた。 競争相手がいない環境で最大の収量を出すように設計されている。 雑草と共存する能力は、品種改良の過程で失われている。

福岡正信自身も、無除草を実現するまでに数十年の試行錯誤を経ている。 彼の畑が「何もしなくても」うまくいったのは、 数十年かけて土壌微生物の生態系が成熟した結果であり、 また、その土地と気候に合った作物の選定を 長年の観察から見出した結果だ。 最初から「何もしない」で成功したわけではない。

川口式の限界——「刈って敷く」はスケールしない

自然農法の実践者として知られる川口由一氏の方法は、 草を刈って敷くことで雑草を管理する。 根は抜かないから土壌は守られる。 刈った草がマルチとなって地表を覆い、新たな雑草の発芽を抑える。 理にかなった方法だ。

しかし、この方法には致命的な制約がある。 手間がかかりすぎる。

川口式の労働集約性：
草が伸びる → 鎌で刈る → 野菜の周りに敷く
→ 数週間後にまた伸びる → また刈る
夏場は雑草の成長が爆発的 → 刈る頻度が激増
→ 一人で管理できる面積が極めて限定される
→ 自給はできても、それ以上の面積は無理

川口式は「一人の人間が、小さな畑で、丁寧に暮らす」ためのモデルとしては美しい。 しかし、肥料危機に備えて食料生産のレジリエンスを構築するという目的には、 面積の制約が致命的だ。 一人が管理できる面積が限られている限り、社会全体の食料安全保障には寄与しない。

二つの自然農法の構造的な違い： 川口式：一株一株の世話。刈って敷く。労働集約的。美しいが小規模。 福岡式：生態系全体の設計。手を出さない。管理ゼロを目指す。スケール可能。 川口式は「植物単位」の管理。 福岡式は「生態系単位」の管理。 スケールの問題を解決するには、福岡式の「管理しない」方向に進むしかない。

福岡正信の答え——すでに実証されている

実は、福岡正信自身がこの問いに対する答えを、 『無III 自然農法』の中で写真と共に示している。

写真33：「何もしないで出来るカンピョウや南瓜」 雑草の中で、カンピョウ（干瓢）や南瓜（かぼちゃ）が育っている。 これらはまさに「雑草に勝てる作物」だ。 大きな葉を広げ、ツルを伸ばし、地面を覆って光を奪い返す。 「何もしない」で収穫できている。

写真34：「雪の中の雑草より強い野菜」 雪の下から、大きな葉を広げた野菜が見える。 冬、雑草が弱る季節に、野菜が優位に立っている。 これが「タイミング戦略」の実証だ。 冬野菜は、雑草との競争を避けて育つ。

写真35：「荒地で野草化した大根やカブ」 これが最も重要な写真だ。 大根やカブが「野草化」している——つまり、 人間の管理なしに、雑草と共存して自生している。 品種改良で失われた「野性」を取り戻した野菜だ。 これこそが福岡正信の到達点であり、無除草の最終形態だ。

福岡正信が示した三つの戦略：
戦略1：雑草に勝てる作物を選ぶ → カンピョウ、南瓜（写真33）
戦略2：雑草が弱い季節に育てる → 冬野菜（写真34）
戦略3：野菜を「野草化」させる → 大根、カブの自生（写真35）

注目すべきは戦略3だ。 「雑草に勝てる野菜を選ぶ」のではなく、 「野菜を雑草のように強くする」。 大根やカブが世代を重ねて自然淘汰される中で、 雑草と共存できる個体だけが生き残り、種を残す。 数世代で、その土地に適応した「野草化した野菜」が生まれる。

これは品種改良の逆だ。 現代農業は「管理された環境で最大の収量を出す」方向に品種改良してきた。 福岡正信は「管理されない環境で自生できる」方向に野菜を戻した。

では、どうするか

問いを整理しよう。

川口式の「刈って敷く」はスケールしない。 しかし福岡式の「何もしない」では野菜が雑草に負ける。 この二つの間にある解を見つけなければならない。

構造的に考えれば、方向は一つだ。 「野菜を雑草から守る」のではなく、 「野菜が雑草に勝つ仕組みを設計する」。 一株ずつ手をかけるのではなく、 生態系のレベルで競争条件を変える。

1. 雑草に勝てる作物だけを選ぶ 繊細な葉物野菜を雑草の中で育てるのは構造的に無理がある。 かぼちゃ・ズッキーニ（大葉で地面を覆って光を奪い返す）。 トウモロコシ・オクラ（背が高く雑草の上に出る）。 サツマイモ（ツルで地表を制圧する）。 まず「雑草に勝てる野菜」だけで始める。 勝てない野菜は、無除草では作らないと割り切る。

2. 果樹・多年草に移行する 一年草の野菜は、毎年ゼロから雑草と競争しなければならない。 これが構造的に不利な原因だ。 果樹や多年草の作物——オリーブ、柑橘類、アーティチョーク、アスパラガス、 ベリー類——は、一度根付けば毎年雑草より優位に立てる。 福岡正信の果樹園がうまくいっていたのは、この構造による。 一年草の畑よりも果樹園の方が、無除草との相性が圧倒的に良い。

3. 被覆作物で「雑草の席を先に埋める」 畑に空席があれば、雑草が座る。 空席をなくせばいい。 クローバー、ヘアリーベッチなどの被覆作物で地面を先に覆い、 その中に作物を植える。 「望ましい植物」で「望ましくない雑草」の席を奪う。 これは除草ではない。生態系の設計だ。

4. タイミングで先行する 雑草にも季節がある。早春、雑草がまだ弱い時期に作物を先行させる。 作物が先に根を張り、先に葉を広げれば、 後から来る雑草に対して優位に立てる。 福岡正信の粘土団子（seed ball）も、このタイミング戦略の一つだ。 秋に冬野菜の種を蒔けば、春の雑草が来る前に根を張れる。

5. AIで「勝てる組み合わせ」を蓄積する どの作物が、どの雑草に対して、どの季節に、どの土壌条件で勝てるのか。 この組み合わせは膨大で、一人の人生では試しきれない。 AIによる画像記録と分析で、「この条件ならこの作物が雑草に勝つ」 というパターンを蓄積することが、 無除草を「哲学」から「再現可能な技術」に転換する鍵だ。

6. 野草化——野菜を雑草のように強くする 福岡正信が『無III 自然農法』の写真35で示した最終戦略。 大根やカブを雑草の中で自然に生えさせ、 実った種をそのまま落として翌年に繋げる。 雑草に負ける個体は淘汰され、勝てる個体だけが種を残す。 数世代で、その土地の雑草と共存できる「野草化した野菜」が生まれる。 これは品種改良の逆だ。 現代農業が「管理された環境で最大収量」を目指したのに対し、 福岡は「管理されない環境で自生できる」方向に野菜を戻した。 最も時間はかかるが、最も手間がかからない。究極の無除草だ。

共通するのは、「手間をかけて雑草を管理する」のではなく、 「手間をかけなくても野菜が勝つ条件を見つける」というアプローチだ。 川口式が「戦術」だとすれば、福岡式は「戦略」を求めている。 そして戦略6の「野草化」は、戦略を超えた——自然そのものへの回帰だ。

1〜5は今日から始められる。 6は数年かかる。しかし、福岡正信はすでに実証している。 このサイトの「畑の記録」は、 まさにその蓄積のためにある。

構造的に正しいことと、実践が容易であることは別だ。

今すぐ始めなければならない理由

土壌微生物の回復には1〜3年かかる。 危機が起きてから始めたのでは遅い。

製油所の破壊は仮説ではない。 ホルムズ海峡の緊張、ウクライナでの製油所攻撃、 紅海の航行リスク——いずれも現在進行形のリスクだ。

化学肥料が手に入らなくなる日が来ることを前提に、 今日から土壌を再生し始めること。 それが、構造分析から導かれる唯一の合理的な結論だ。

結論： 化学肥料依存の農業は、石油精製という単一障害点を持つ脆弱なシステムだ。 30年間の気候変動議論は順序が逆で、土壌と生物多様性が先だった。 自然農法は理想論ではなく、外部依存を排除したサバイバル戦略である。 土壌の回復には時間がかかる。今日始めなければ、危機に間に合わない。
