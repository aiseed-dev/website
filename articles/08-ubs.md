## 金融アナリストが見ない因果連鎖

金融の世界では、リスクを「金融変数」で分析する。
金利、為替、信用スプレッド、VIX——
すべて金融市場内部の数字だ。

しかし、金融危機のトリガーは金融市場の外にある。
物理的な事象——製油所の破壊、港湾の封鎖、パイプラインの損傷——が、
金融システムに伝播する経路が存在する。

この経路を追跡する。

## 6つのステップ：製油所から不動産ファンドへ
1
製油所の破壊 → エネルギー供給の途絶
ホルムズ海峡周辺の製油所が物理的に破壊される。
封鎖ではなく破壊。修復に3〜5年。
世界の石油精製能力が急落する。
原油は存在しても、精製品（ガソリン、軽油、ナフサ、硫黄）が供給できない。
2
エネルギー価格の急騰 → コストプッシュ・インフレ
精製品の供給不足は、エネルギー価格を直撃する。
ガソリン、軽油、ジェット燃料が高騰。
輸送コストが上昇し、すべての商品価格に転嫁される。
食料は肥料不足（硫黄トラップ）と輸送コスト上昇の二重打撃を受ける。
コストプッシュ・インフレが世界同時に発生。
3
中央銀行の利上げ → 金利の急上昇
インフレに対して中央銀行は利上げで対応する。
FRB、ECB、日銀——すべてが同時に金融引き締めに向かう。
しかし、コストプッシュ・インフレに対して利上げは本質的に無力だ。
供給不足が原因であり、需要を冷やしても供給は回復しない。
それでも中央銀行は利上げする。他に手段がないからだ。
4
金利上昇 → 不動産価格の下落
不動産は金利に最も敏感な資産クラスだ。
住宅ローン金利が上昇すれば、購入可能額が減り、需要が萎縮する。
商業不動産は、テナント企業の業績悪化と金利上昇の二重圧力を受ける。
不動産価格が20〜40%下落するシナリオが現実的になる。
5
不動産下落 → ファンドの流動性危機
ここでUBSが登場する。
UBSは世界最大級の不動産ファンドを運用している。
Credit Suisseの買収により、不動産エクスポージャーはさらに拡大した。
不動産価格が下落すれば、ファンドの評価額が下がる。
投資家が解約を申し出る。しかし不動産は即座に売却できない。
流動性のミスマッチ——流動的な負債（投資家の解約請求）と非流動的な資産（不動産）。
6
流動性危機 → システミック・リスク
一つのファンドの解約停止は、他のファンドへの連鎖を引き起こす。
「あのファンドが解約停止なら、うちも早く解約しよう」——パニックの伝播。
銀行間の信用が収縮する。短期金融市場が機能不全に陥る。
2008年のリーマン・ショックと同じ構造が、物理的トリガーから始まる。

## なぜ金融アナリストはこの連鎖を見ないのか

金融アナリストは、ステップ3以降——金利、不動産価格、ファンド流動性——は分析できる。
しかし、ステップ1と2——製油所の物理的破壊とその波及——は分析の射程に入っていない。

:::highlight
**分野の壁：**
エネルギーアナリストは製油所と原油価格を見る。金融アナリストは金利と信用を見る。農業アナリストは作物と肥料を見る。
しかし、製油所 → 肥料 → 食料 → インフレ → 金利 → 不動産 → ファンド という連鎖は、**誰の専門分野にも属さない。**
構造的思考だけが、この分野横断的な因果連鎖を追跡できる。
:::

## UBSの具体的な脆弱性

なぜUBSを名指しするのか。構造的な理由がある。

2023年、UBSはCredit Suisseを救済買収した。
この買収により、UBSはCredit Suisseの問題資産を引き継いだ。
その中には、商業不動産への大規模なエクスポージャーが含まれている。

:::chain
**UBSの脆弱性の構造：**
Credit Suisse買収 → 問題資産の引き継ぎ
不動産エクスポージャーの拡大 → リスク集中
スイスの「大きすぎて潰せない」問題 → スイスGDPの2倍の資産規模
スイス政府の救済能力の限界 → システミック・リスク
:::

スイスのGDPは約8,000億ドル。UBSの総資産はその約2倍。
UBSが流動性危機に陥った場合、スイス政府が単独で救済することは事実上不可能だ。
これは2008年のAIGやリーマン・ブラザーズと同じ構造——
一つの金融機関が、母国の救済能力を超える規模に膨張している。

## 物理的因果関係を追跡する意味

このシリーズ全体を通じて、同じ方法論を使ってきた。
生産ルートを追跡する。物理的プロセスを見る。分野の境界を越える。

:::chain
**5つの記事の接続：**
[01. 農業の間違い](/insights/agriculture/)：化学肥料 → 石油精製 → 単一障害点
[02. 核融合の間違い](/insights/fusion/)：エネルギー代替 ≠ 素材代替 → 硫黄トラップ
[03. AGIの間違い](/insights/agi/)：情報処理 ≠ 物理的作業 → 人間の仕事は増える
[07. NVIDIAの崩壊](/insights/nvidia/)：AGI不採算 → GPU需要消滅
05. UBSと金融崩壊：物理的破壊 → 金融システムへの伝播
**全て同じ構造：**物理的プロセスを追跡し、分野の境界を越えて因果連鎖を可視化する
:::

金融の世界は、自分たちの世界の中だけでリスクを分析している。
しかし、本当のリスクは物理世界から来る。
製油所の破壊は「地政学リスク」としてVIXに少し反映されるだけだ。
その先の連鎖——肥料、食料、インフレ、金利、不動産、ファンド——は、
誰もモデルに組み込んでいない。

:::quote
金融危機のトリガーは金融市場の外にある。
製油所の煙突が崩れた時、不動産ファンドの価値が崩れる。
この因果連鎖は、構造を見る者にしか見えない。
:::

## 構造分析シリーズの結論

5つの記事を通じて、一つの方法論を示した。

:::highlight
**方法論：**
1. 生産ルートを追跡する（原料→加工→製品→消費）
2. 物理的プロセスを見る（エネルギーと素材を区別する）
3. 分野の境界を越える（農業、エネルギー、AI、金融を接続する）
4. 単一障害点を特定する（一点の破壊で全体が崩壊する箇所）
5. 時間軸を追跡する（破壊は瞬時、回復は年単位）
:::

この方法論は、AIの力を借りて誰でも実行できる——
ただし、「何を追跡するか」を知っている人間だけが。

「肥料の生産ルートを調べて」と指示できる人間が使えば、
AIは硫黄トラップを発見する。
「肥料危機はどうなりますか」と聞く人間が使っても、
AIは一般的な解説を返すだけだ。

構造的思考は、AIの前提条件だ。
そして、構造的思考の最も実践的な応用は、
自分の食料を自分で育てられるようにしておくことだ。
