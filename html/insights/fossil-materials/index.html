Structural Analysis 06

化石資源と現代文明

便利な生活の正体は、石油から作られた素材だった。

2025.04.03

Insights — 構造分析シリーズ 06

あなたの生活のほぼ全てが石油でできている

朝起きる。ポリエステルのシーツから体を起こす。 プラスチックの歯ブラシで歯を磨く。 ナイロンの服を着る。合成ゴムの靴を履く。 スマートフォンのポリカーボネート筐体を手に取る。 アスファルトの道路を歩く。 食品をポリエチレンの袋から取り出す。 ポリプロピレンの容器に入った弁当を食べる。

この日常のどこに「石油」があるか。全部だ。

石油から作られる日常の素材：

ナフサ → エチレン → ポリエチレン（レジ袋、ラップ、容器）
ナフサ → プロピレン → ポリプロピレン（食品容器、自動車部品）
ナフサ → ベンゼン → ナイロン、ポリウレタン（衣類、断熱材）
ナフサ → スチレン → ポリスチレン（発泡スチロール、食品トレー）
ナフサ → PET（ペットボトル、ポリエステル繊維）
ナフサ → 塩ビ（配管、窓枠、電線被覆）
重質留分 → アスファルト（道路）
重質留分 → 潤滑油（あらゆる機械の動作）
副産物 → パラフィン（ろうそく、化粧品、医薬品コーティング）

プラスチックは「環境に悪い使い捨て製品」として語られることが多い。 しかし、それは問題の表面にすぎない。 本当の問題は、現代文明のインフラそのものが化石資源の素材機能に依存していることだ。

プラスチック問題の本質は「代替不能性」

「プラスチックをやめよう」「脱プラスチック」—— この議論は、プラスチックを「選択肢の一つ」として捉えている。 紙に変えればいい。竹に変えればいい。ガラスに変えればいい。

しかし、構造的に見れば、プラスチックが代替不能な用途が大量にある。

用途 なぜプラスチックなのか 代替の現実
医療器具（注射器、チューブ、血液バッグ） 滅菌可能、軽量、使い捨てで感染防止 ガラスは重く割れる。金属は高コスト。現実的な代替なし
食品包装 気密性、軽量、低コスト、食品との非反応性 紙は水に弱い。ガラスは重く輸送コスト増。部分的代替のみ
電線被覆 絶縁性、柔軟性、耐候性 ゴムは劣化が速い。代替なし
水道管・下水管 腐食しない、軽量、長寿命 金属管は腐食する。陶管は重く脆い。大規模代替は非現実的
自動車部品 軽量化（燃費向上）、成形の自由度 金属で全て作ると重量・コスト増大
半導体のフォトレジスト 微細パターン形成に不可欠 代替なし。半導体製造が停止する

「脱プラスチック」は、レジ袋やストローの話をしている。 しかし、医療器具、電線、水道管、半導体のプラスチックをやめたら、 文明が止まる。

プラスチック問題の構造：

表面の問題：使い捨てプラスチックの環境汚染（レジ袋、ストロー）
本質の問題：現代文明のインフラがプラスチックなしでは機能しない
見えていない問題：そのプラスチックの原料（ナフサ）は石油精製から来る

大企業が存在できるのは化石資源があるから

現代の大企業——Amazon、Apple、Toyota、Walmart——は、 化石資源のインフラの上に成り立っている。

大企業を支えるインフラの化石資源依存：

物流：軽油（トラック・船舶）+ アスファルト（道路）+ プラスチック（梱包材）
製造：ナフサ（部品素材）+ 潤滑油（機械動作）+ 合成ゴム（シール・ベルト）
通信：塩ビ（電線被覆）+ フォトレジスト（半導体）+ 光ファイバー被覆
食品：ポリエチレン（包装）+ 硫黄（肥料）+ ナフサ（農薬原料）
建設：アスファルト + 塩ビ配管 + 断熱材（ポリウレタン）+ 塗料

Amazonの翌日配送は、アスファルトの道路とプラスチックの梱包材がなければ成立しない。 Appleの製品は、半導体のフォトレジストとポリカーボネートの筐体がなければ作れない。 Toyotaの車は、合成ゴムのシールと潤滑油がなければ動かない。

大企業は「テクノロジー企業」や「製造企業」を自称するが、 構造的には「化石資源の加工業」だ。 化石資源の素材機能がなくなれば、これらの企業のビジネスモデルは崩壊する。

エネルギー転換後に残る「素材の空白」

再生可能エネルギーと核融合が完全にエネルギー問題を解決したとしよう。 石油を燃やす必要はなくなった。CO2排出はゼロになった。

しかし、石油を燃やさなくなった世界で、 ナフサはどこから来るのか。

エネルギー転換のパラドックス：

石油をエネルギーとして燃やさなくなる → 石油精製の経済性が崩壊する → 精製の副産物（ナフサ、硫黄、アスファルト）も供給されなくなる → 素材としての石油も手に入らなくなる

エネルギー転換は、エネルギー問題を解決すると同時に、 素材問題を悪化させる可能性がある。

現在、石油の約14%が化学原料（素材用途）として使われている。 残りの86%がエネルギー用途だ。 エネルギー用途がなくなれば、素材用途のためだけに石油を精製することになる。 そのコストは現在の何倍にもなる。 なぜなら、精製コストを86%のエネルギー用途で分担していたからだ。

化石資源の後に来るもの——微生物と植物

化石資源が枯渇するか、エネルギー転換で精製が止まるか—— いずれにせよ、ナフサベースの素材が手に入らなくなる時代が来る。

その時、素材はどこから来るのか。 答えは二つしかない。微生物と植物だ。

微生物由来の素材

バイオプラスチック——微生物が糖を発酵させてPHA（ポリヒドロキシアルカノエート）を生産。生分解性。 バイオ繊維——細菌セルロース。ナタデココと同じ原理。 バイオ接着剤——菌糸体ベースの接着剤。 発酵化学品——エタノール、乳酸、コハク酸。化学原料として使用可能。

植物由来の素材

セルロース——木材、麻、竹から。紙、繊維、建材。 リグニン——木材の副産物。接着剤、炭素繊維の原料候補。 天然ゴム——ゴムの木から。合成ゴムの代替。 植物油——亜麻仁油、桐油。塗料、潤滑剤。 デンプン——生分解性プラスチックの原料。

菌糸体（マイセリウム）

キノコの根に当たる菌糸体は、素材革命の鍵を握る。 断熱材、梱包材、建材、皮革代替品として既に実用化が始まっている。 農業廃棄物を基質として成長し、生分解性。 発泡スチロールの直接的な代替候補。

藻類

微細藻類はバイオ燃料だけでなく、バイオプラスチック、 食品添加物、化粧品原料、飼料を生産できる。 CO2を吸収しながら成長し、耕作地を必要としない。 水と光とCO2があれば、どこでも培養可能。

これらのバイオ素材は、現時点では石油由来のプラスチックに コスト・性能の両面で劣る。 それは事実だ。

しかし、石油由来の素材が手に入らなくなれば、 「劣る」かどうかは問題ではなくなる。 それしかないからだ。

福岡正信が見ていた未来

福岡正信は1970年代に「わら一本の革命」を書いた。 石油危機の直後だ。 しかし、彼が批判したのはエネルギー依存ではなく、 化学肥料と農薬への依存——つまり、石油の素材機能への依存だった。

自然農法は、石油文明が終わった後の農業モデルではない。
石油文明が始まる前から存在していた農業モデルだ。
微生物と植物が素材を供給し、人間はその中で暮らしていた。
それが本来の姿であり、戻るべき場所だ。

リジェネラティブ農業は、単に「化学肥料を使わない」農業ではない。 土壌微生物を育て、菌根菌のネットワークを再構築し、 植物と微生物の共生系を回復させる—— これは、化石資源後の世界で素材を供給するシステムの基盤でもある。

構造的に見える未来

化石資源後の世界は、「不便になる」世界ではない。 素材の供給源が変わる世界だ。

素材供給の転換：

石油精製 → ナフサ → プラスチック → 大量生産・大量廃棄

微生物発酵 → バイオポリマー → 生分解性素材
植物栽培 → セルロース・リグニン → 繊維・建材
菌糸体培養 → マイセリウム → 梱包材・断熱材
藻類培養 → バイオオイル → 化学原料

この転換には、膨大な量の「生物学的インフラ」が必要だ。 微生物の培養施設、植物の栽培システム、菌糸体の生産拠点—— これらは全て、土壌の健康と生物多様性の上に成り立つ。

つまり、化石資源後の世界の準備は、 土壌の再生から始まる。

結論：

現代文明の便利さの正体は、化石資源から作られた素材だ。 エネルギーは代替できるが、素材の代替には微生物と植物が必要になる。 大企業も、日常の便利さも、化石資源の素材機能の上に成り立っている。 化石資源後の世界に備える唯一の方法は、 微生物と植物を育てる力を取り戻すこと—— すなわち、土壌を再生することだ。
