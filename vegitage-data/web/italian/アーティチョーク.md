# アーティチョーク

*Cynara cardunculus* var. *scolymus* — 地中海の「食用の花」

アーティチョークは、キク科（Asteraceae）に属する多年草で、未熟な花のつぼみ（花序）を食用にする「花菜」です。地中海西部から北アフリカ原産の野生種カルドンを祖先に、棘の多い野草から数千年かけて選抜され、食卓を飾る食材へと姿を変えてきました。

イタリアはこの「食用の花」の遺伝的多様性の中心地であり、世界有数の生産国です。ラツィオの **Carciofo Romanesco del Lazio IGP**、サルデーニャの **Carciofo Spinoso di Sardegna DOP** をはじめ、地域ごとに固有の品種体系（DOP/IGP）が発達しました。

栽培は地中海性気候に適応し、加熱調理や保存食として広く使われます。近年は気候変動と労働力不足が生産の課題となっています。
