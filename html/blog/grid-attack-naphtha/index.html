もしトランプ大統領が4月6日にイランの発電所を破壊したら

「48時間以内に合意するか海峡を開け。さもなくば地獄が降り注ぐ」——実は世界に地獄が降り注ぐ

2026.04.05

構造分析ノート

最近、トランプ大統領はイランに対して強硬な発言を行っている。

トランプ大統領の発言

  • 3月22日：「48時間以内にホルムズ海峡を完全に開かなければ、イラン最大の発電所から破壊する」と威嚇 (CBS News)
  • 3月23日：「非常に生産的な対話」があったとして攻撃を5日間延期 (日本経済新聞)
  • 3月26日：攻撃期限を4月6日午後8時（米東部時間）まで再延期 (Al Jazeera)
  • 3月30日：「合意に至らなければ、イランの発電所、油井、ハーグ島、そして淡水化プラントも全て吹き飛ばす」とTruth Socialに投稿 (Al Jazeera)。アムネスティ・インターナショナルは「戦争犯罪の威嚇」と非難 (Amnesty International)
  • 4月1日：国民向けテレビ演説で「戦争は完了に近づいている」「今後2〜3週間で極めて激しく攻撃する」「石器時代に戻す」と発言 (CNBC, Al Jazeera)
  • 4月3日：「米国はイランの残りの破壊をまだ始めてすらいない。次は橋、その次は発電所だ！」とTruth Socialに投稿 (CNBC)
  • 4月4日：「48時間以内に合意するか海峡を開け。さもなくば地獄が降り注ぐ」（"Time is running out — 48 hours before all Hell will reign down on them"）(Axios)

パキスタン、トルコ、エジプトによる間接交渉が行われているが、イランは一時停戦を拒否し、恒久的な戦争終結と再攻撃しない保証を要求している。

期限は4月6日午後8時（米東部時間）——日本時間4月7日午前9時。あと2日を切った。

ブレント原油は2月28日以降55%以上上昇し、126ドル/バレルに達した。国際エネルギー機関（IEA）は「史上最大のエネルギー安全保障上の危機」と評した (Economic impact — Wikipedia)。

多くの人はこれを「中東の戦争」として見ている。

違う。これは世界のサプライチェーンの危機だ。

4月6日（日本時間4月7日）——発電所を攻撃したら何が起きるか

交渉は進展していない。トランプ大統領がイランの発電所を攻撃した場合、何が予想されるか。

イランの報復はエスカレートする

イランは3月30日の時点で、「イラン沿岸領土への攻撃は、全ての湾岸航路への機雷敷設を引き起こす」と警告している。現在の機雷はホルムズ海峡に集中しているが、発電所攻撃があればペルシャ湾全域に拡大する可能性がある。

イランはなお80〜90%の小型艦艇と機雷敷設能力を保持している。数百個の機雷を短期間で敷設できる。

湾岸の発電所・淡水化設備が報復対象になる

これまでイランは湾岸の石油施設と淡水化設備を攻撃してきた。トランプがイランの発電所を攻撃すれば、「同等の報復」として、湾岸諸国の発電所・淡水化設備をさらに大規模に攻撃する口実を与える。

トランプがイランの発電所を破壊する → イランが湾岸の発電所・淡水化設備を破壊する

これは報復の連鎖だ。そして湾岸諸国は淡水化を発電に依存している。電力が止まれば水も止まる。

機雷封鎖はさらに長期化する

機雷の掃海は極めて時間がかかる作業だ。1隻ずつ、1海里ずつ、機雷を探知して除去していく。戦時中は掃海艦自体が攻撃対象になる。イランが全湾岸航路に機雷を敷設すれば、封鎖の解除には年単位の時間がかかる。

施設復旧は封鎖で阻まれる

破壊された精製施設の復旧には3〜5年かかる。しかし海峡が封鎖されている限り、復旧に必要な資材や機材も搬入できない。封鎖と施設破壊は互いを悪化させる。

淡水化設備が壊れると人が住めなくなる

湾岸諸国の致命的な弱点は、飲料水を海水淡水化に依存していることだ。

すでにバーレーンの淡水化プラントがドローンで損傷し（3月8日）、クウェートの発電・淡水化プラントがミサイルで被弾して作業員1名が死亡している（3月30日）。4月3日にもクウェートの淡水化プラントが再び攻撃された。

湾岸6カ国は合計6,200万人の飲料水を淡水化に依存している。発電所攻撃による報復で淡水化設備が大規模に破壊されると：

淡水化設備破壊 → 飲料水がなくなる → 人が住めなくなる → 施設の管理・復旧ができない → 石油生産の再開が不可能に

復旧には厳密な順序がある。まず機雷を撤去しなければ、資材を運ぶ船が入れない。次に発電所と淡水化設備を建設しなければ、人が住めない。人が住めて初めて、精製施設の復旧に着手できる。

機雷撤去 → 発電所・淡水化設備の建設 → 精製施設の復旧。

この順番は飛ばせない。淡水化設備がなければ人が住めず、人が住めなければ何も始まらない。各段階に年単位の時間がかかるため、復旧は十年単位になる可能性がある。

6,200万人の難民

飲料水がなくなれば、人は数日で死ぬ。淡水化設備の大規模な破壊が起きた場合、湾岸6カ国の住民に残された選択肢は一つしかない。逃げることだ。

サウジアラビア3,600万人、UAE1,000万人、クウェート470万人、カタール290万人、バーレーン170万人、オマーン470万人。合計6,200万人。このうちの多くが外国人労働者だ——UAE人口の約88%、カタール人口の約85%が外国人である。彼らにはそもそも「留まって耐える」理由がない。

淡水化設備破壊 → 飲料水消失 → 数日〜数週間で居住不能 → 6,200万人規模の難民危機

これは仮定の話ではない。すでにバーレーンとクウェートの淡水化プラントが攻撃されている。トランプがイランの発電所を攻撃すれば、報復として湾岸の淡水化設備がさらに大規模に破壊される。

行き先はどこか。陸路で逃げられるのはサウジアラビア経由のヨルダン、イラク、あるいは海路でインド亜大陸。しかし空港も港も機能しているとは限らない。航空ハブのドバイ、ドーハ、アブダビはすでに軍事作戦と空域制限で航空キャパシティが26%減少し、3,400便以上がキャンセルされている。

2015年のシリア難民危機は約560万人だった。 湾岸の人口は6,200万人。

淡水化設備の破壊が大規模に起きれば、シリア難民危機の10倍以上の規模の人道危機が発生する。受け入れ先の国もまた、エネルギー危機と食料危機の渦中にある。

世界はこの規模の難民を受け入れる能力を持っていない。しかも難民が流出する先の国々——インド、パキスタン、ヨルダン、エジプト——自身がホルムズ海峡封鎖によるエネルギー・食料危機の直撃を受けている。危機が危機を増幅する構造だ。

その間、油田はどうなるか

復旧を待つ間も、油田は劣化し続ける。油田は放置すれば自然に回復する資源ではない。常に管理し続けなければ、生産能力そのものが失われる。

  • 坑井の崩壊 — ケーシング（鋼管）は腐食性ガス（硫化水素）と高温にさらされ続ける。メンテナンスが止まれば、坑井が崩壊し、掘り直しが必要になる
  • 貯留層への水・ガス浸入 — 圧力制御が止まると、水やガスが油層に侵入する。一度侵入すると、元の生産性には戻らない
  • 地上設備の劣化 — パイプライン、ポンプ、分離装置は砂漠環境で急速に腐食する。電力が途絶えた状態では防食措置も機能しない
  • 二次回収・三次回収システムの停止 — 湾岸の成熟油田の多くは、水攻法やガス圧入で生産量を維持している。これらが止まれば、油層の圧力が不可逆的に低下し、再開後も以前の生産量には戻らない

施設を直しても、油田が同じ量を産出するとは限らない。

放置期間が長いほど、回復不能な損傷が蓄積する。復旧に10年かかるなら、その間に失われる生産能力は永久に戻らない可能性がある。

何が失われるか

湾岸6,200万人の生活基盤

淡水化設備と発電所が破壊されれば、湾岸6カ国の6,200万人が飲料水と電力を失う。これは軍事施設への攻撃ではない。民間インフラの壊滅だ。

サウジアラビア、UAE、クウェート、カタール、バーレーン、オマーン——これらの国は淡水の大半を海水淡水化に依存している。電力がなければ淡水化は動かない。水がなければ人は住めない。

発電所攻撃は、電力の問題ではない。6,200万人の飲料水の問題だ。

世界の食料生産

エネルギー危機は食料危機に直結する。

ナフサ / 天然ガス → 水素 → アンモニア → 窒素肥料
窒素肥料 → 世界の食料生産の約50%を支えている

湾岸地域は世界で最も広く使われる窒素肥料「尿素」の世界貿易の**46%**を供給している。インド（輸入依存度18%）、ブラジル（同10%）など世界の主要農業国が直撃を受けている。

さらに見落とされている事実がある。湾岸諸国は**世界の硫黄供給の約45〜50%**を占めている。硫黄は石油精製の副産物だ。第11章「規制の再設計」で分析した通り、リン酸肥料は硫酸なしに作れない。

湾岸の石油生産壊滅 → 硫黄の副産物もなくなる（世界の45〜50%）→ 硫酸不足 → リン酸肥料の生産に影響

窒素肥料とリン酸肥料が同時に危機に陥る。第11章で警告した「ピーク・サルファー」が、脱炭素ではなく戦争によって現実化した。

これは「ガソリンが高くなる」問題ではない。世界の食料生産を支える肥料の原料が消える問題だ。

自然農法だけが影響を受けない

化学肥料に依存しない農業だけが、この構造的リスクから自由だ。

福岡正信の自然農法、Christine Jones博士のLight Farming。不耕起、無肥料、無農薬。土壌微生物との共生による窒素固定。

ホルムズ海峡が閉まっても、土壌微生物は止まらない。

構造分析の結論は常に同じ場所に収束する。

自然から離れるほど、生きるコストは上がる。イラン戦争は、そのコストが一瞬で跳ね上がることを証明した。
