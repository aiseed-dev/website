電力網が止まると、食料が消える

トランプの電力網攻撃が意味すること

2026.04.05

構造分析ノート

電力網攻撃は「停電」ではない

トランプ政権による電力網への攻撃——多くの人はこれを「停電」の問題だと思っている。

違う。

電力網が止まることの本当の意味は、製油所が止まることだ。

製油所が止まると何が起きるか

原油の蒸留は連続プロセスだ。電力が止まれば、蒸留塔は停止する。

蒸留が止まると、全ての留分が同時に止まる。

原油 → 蒸留塔（電力必要）→ ガソリン + ナフサ + 灯油 + 軽油 + 重油
電力停止 → 全て停止

第2章「化石資源と素材」で説明した通り、原油の蒸留は「選べない」。ガソリンだけ作ることも、ナフサだけ作ることもできない。全てが同時に出てくるか、全てが止まるかだ。

ナフサが止まると肥料が止まる

ナフサは石油化学の原料だ。そしてナフサから作られるものの中に、アンモニアがある。

ナフサ → エチレン / プロピレン → プラスチック、合成繊維
天然ガス or ナフサ → 水素 → アンモニア → 窒素肥料

世界の食料生産の約半分は、化学肥料に依存している。その化学肥料の原料は、化石資源だ。

電力網攻撃 → 製油所停止 → ナフサ停止 → 肥料原料停止 → 食料生産に影響

これは「停電で不便」の問題ではない。食料供給の構造的リスクだ。

日本の脆弱性

日本は特に脆弱だ。

  • ナフサの大半を輸入に依存
  • 肥料原料（リン、カリウム）もほぼ全量輸入
  • 食料自給率は38%（カロリーベース）

電力網攻撃が起きる国から輸入している場合、その国の電力問題は日本の食料問題になる。

日本の食料は、他国の電力網に依存している。

これが構造分析の結論だ。停電の問題ではない。食料の問題だ。

構造は繋がっている

第1章で指摘した通り、化石資源の枯渇は気候変動より先に来る。

第3章で示した通り、化学肥料が消えれば慣行農業は成り立たない。

第11章で分析した通り、脱炭素が硫黄供給を止め、リン酸肥料も止まる。

電力網攻撃は、この構造を一瞬で現実にするシナリオだ。

化石資源の枯渇は数十年後に来る。しかし電力網攻撃は明日来る可能性がある。

自然農法が保険になる

第3章とLight Farmingで繰り返し述べてきたこと——

化学肥料に依存しない農業だけが、この構造的リスクから自由だ。

福岡正信の自然農法、Christine Jones博士のLight Farming。不耕起、無肥料、無農薬。土壌微生物との共生。

これは理想論ではない。電力網が止まっても食料を生産できる唯一の方法だ。

構造分析の結論は常に同じ場所に収束する。

自然から離れるほど、生きるコストは上がる。電力網攻撃は、そのコストが一瞬で跳ね上がるシナリオだ。
