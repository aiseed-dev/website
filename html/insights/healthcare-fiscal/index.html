社会の設計ミス

軍需産業、IT企業、デスクワーカー、医療、年金——全てが同時に崩壊する

2025.04.04

構造分析シリーズ 09

軍需産業の衰退

このシリーズで見てきた構造を、産業の未来に当てはめる。 最初に崩壊するのは軍需産業だ。

軍需産業の化石資源依存： ジェット戦闘機 → ジェット燃料（灯油留分）＋複合材料（石油化学） 戦車 → 軽油＋合成ゴム＋高機能プラスチック 軍艦 → 重油＋塗料（石油系）＋配線被覆（PVC） ミサイル → 固体推進剤の一部は石油化学由来＋電子部品（フォトレジスト） 兵站（ロジスティクス） → 軽油で動くトラックと輸送船が全て

第4章で見たように、ドローン＋AIが従来の大型兵器を無力化しつつある。 そして、大型兵器は化石資源なしには動かない。

軍需産業衰退の構造：
ドローン＋AIが大型兵器を無力化 → 戦闘機・戦車の需要減少
化石資源の価格上昇 → 大型兵器の運用コスト上昇
化石資源の枯渇 → 大型兵器の製造自体が困難に
→ ロッキード・マーティン、レイセオン、BAEシステムズ——
→ 従来型軍需産業の市場が構造的に縮小する

軍需産業で働く数百万人の雇用が影響を受ける。

IT企業衰退の構造：
AGIが採算に合わない（第6章）→ AI投資の縮小
→ NVIDIAのGPU需要が消える（第7章）
→ データセンター建設の減速
→ クラウド事業の成長鈍化（AWS、Azure、GCP）

SIer税の崩壊（第8章）→ システムインテグレーター業界の縮小

影響を受ける企業： NVIDIA → GPU需要の急減 Oracle → PostgreSQL移行でライセンス収入消滅 Microsoft → Office/Azure/Windows税の段階的崩壊 Salesforce → Claude＋自前CRMへの移行 Accenture、NTTデータ等のSIer → Claudeが工数の大半を代替 大手コンサルティングファーム → Claudeが構造分析を代替

デスクワーカーの激減

軍需産業とIT企業の衰退は産業レベルの話だ。 しかし、より広範な影響が企業の内部で起きる——デスクワーカーの激減。

第6章で見たように、AGIは不要だ。 現在のAI（Claude）で、デスクワークの大半は代替できる。

デスクワーカー減少の規模： 日本の就業者約6,700万人のうち、事務従事者は約1,300万人。 このうち、AIで代替可能な定型業務は半数以上と推定される。 数百万人規模のデスクワーカーが「不要」になる時代が来る。 これは10年、20年先の話ではない。現在のAIで既に可能だ。

これがUBI（ベーシックインカム）の議論に繋がる。 しかし第6章で示したように、UBIは間違いだ。 バイオ素材の製造、土壌の再生、生態系の回復—— 物理的な仕事は永久に人手が必要だ。 デスクワーカーは消えるが、仕事はなくならない。変わるだけだ。

大都市から土地のある場所へ——人口の大移動

軍需産業、IT企業、デスクワーカー—— これらの産業から溢れた人間は、どこへ行くのか。

第2章で見たように、化石資源の後にはバイオ素材が来る。 第3章で見たように、化学肥料なしの食料生産が必要になる。 第6章で見たように、バイオ素材の製造には永久に人手が必要だ。

仕事はある。しかし、その仕事は大都市ではできない。

バイオ素材・食料生産に必要なもの： 木材（建材・繊維・化学原料） → 林地の管理と伐採・植林が必要 植物原料の栽培 → 農地が必要 リジェネラティブ農業 → 広大な農地と放牧地が必要 野菜の栽培実践 → 圃場が必要（実践者が試行錯誤する場） 微生物・菌糸体の培養 → 土地とスペースが必要 バイオマスの収集 → 農村部・山林での物理的労働 東京のオフィスビルの中では、何一つできない。

人口移動の構造：
軍需産業の衰退 → 都市部の防衛関連雇用が消える
IT企業の衰退 → 都市部のオフィスワーカーが溢れる
デスクワーカーの激減 → 都市部に人を集める理由がなくなる
→ 同時にバイオ素材・食料生産の人手が足りない
→ これらの仕事は土地がないとできない
→ 人口が大都市から地方へ移動する

考えてみれば、日本の地方の町は既にこの規模で存在している。 商店がある。学校がある。診療所がある。農地と林地に囲まれている。 大きく変わるのは地方ではない。東京だ。

現在 転換後
--- --- ---
東京圏 3,600万人がオフィスに通勤 集中する理由が消える
地方の町 過疎化が進行 人が戻ってくる（受け皿は既にある）
主な雇用 デスクワーク、IT、サービス業 バイオ素材製造、食料生産、林業、土壌再生
必要な資源 オフィス、データセンター、電力 農地、林地、水、日照、土壌

東京一極集中は、化石資源とデスクワークの時代の産物だ。 デスクワークが消え、仕事が土地ベースに変われば、東京に3,600万人が住む理由がなくなる。

地方の暮らし自体はそれほど変わらない。 町があり、サービスがあり、農地と林地がある——その構造はそのまま受け皿になる。 変わるのは、東京一極集中が解消されることだ。

ただし、受け皿があるからといって、人が自然に移動するわけではない。 政策的な後押しが必要だ。

必要な政策： 農地・林地の取得支援 → 都市部からの移住者が土地を持てる仕組み 職業訓練 → デスクワーカーがバイオ素材製造・農業・林業を学ぶ場 住居の初期支援 → 空き家の活用、移住者向け住居の整備 地方の町のサービス維持 → 診療所・学校・商店を維持する財政措置 実践圃場の整備 → 野菜の栽培実践を地方で行う場。研究者ではなく実践者が必要。分析はAIが行う

そしてもう一つ、最も重要な政策がある。 自由貿易の見直しだ。

今までの自由貿易は、ある前提の上に成り立っていた。

自由貿易の前提：
安い石油 → 安い輸送コスト → 地球の裏側からモノを運んでも採算が合う
→ 人件費の安い国で作り、先進国で消費する
→ 国内の農業・林業は「非効率」として切り捨てられる
→ 完全な自由貿易が正しいとされた

この前提自体が、化石資源に依存していた。

国内でバイオ素材・食料・木材の生産体制を作っても、 完全な自由貿易のままでは、海外からの安い輸入品に潰される。 土地ベースの仕事で暮らす人々の生活が成り立たない。

自由貿易を維持した場合：
国内で農業・林業・バイオ素材の生産を始める
→ 海外から安い農産物・木材が輸入される
→ 価格競争で国内生産者が生活できなくなる
→ 人が地方に留まる理由がなくなる
→ 産業構造の転換自体が失敗する

化石資源が枯渇すれば、いずれ輸送コストは上がり、 遠くから安いものを運ぶモデル自体が成り立たなくなる。 しかし、その前に国内の生産基盤が潰れてしまっては意味がない。 自由貿易の前提が崩れるなら、貿易政策も変えなければならない。

産業構造の転換は避けられない。 しかし、転換が起きてから慌てるのではなく、 今から地方の受け入れ体制と貿易政策を整えておくことが、政策の仕事だ。

地方の過疎化は「解決すべき問題」ではなく、 産業構造の転換が解決する結果になる。 ただし、それを円滑に進めるための政策は必要だ。

現在の医療・年金は、この社会に合わない

人口が分散し、仕事が土地ベースに変わる。 この構造転換に対して、現在の医療・年金制度は根本的にミスマッチだ。

現在の制度の前提：
都市に人口が集中している → 大規模病院・透析施設を都市部に集約
サラリーマンが大半 → 給与から天引きで社会保険料を徴収
65歳で定年退職 → 年金で余生を過ごす
デスクワークで体を壊す → 医療で治す

転換後の社会：
人口が地方に分散 → 都市集約型の医療インフラが届かない
土地ベースの仕事 → 給与天引きモデルが合わない
身体を使う仕事を長く続ける → 「65歳で引退」の前提が消える
自然な食と運動 → そもそも病気になりにくい

現在の制度は、化石資源と都市集中の時代に設計された。 その時代が終わるのに、制度だけが残っている。

透析医療——化石資源で命を繋ぐ構造

日本の透析患者数は2024年末で337,414人。 国民の約367人に1人が透析を受けている。 患者一人あたり年間約480万円。マクロ総額は年間約1.6兆円。

透析医療の化石資源依存：

ダイアライザー（人工腎臓） → ポリスルホン膜＋ポリカーボネート外筒 → 石油化学製品
血液回路・穿刺針 → ポリ塩化ビニル（PVC）チューブ → 石油化学製品
透析液バッグ → プラスチック素材 → 石油化学製品
超純水の精製 → RO装置の常時稼働 → 大量の電力
透析液の加温 → 37度まで加熱 → 大量の電力
14万台のコンソール → 空調・駆動 → 大量の電力

透析治療の63.3%が血液透析濾過（HDF）——従来型より大量の超純水と電力を消費する。 在宅透析はわずか3.4%。世界最低水準だ。 全国4,512施設に148,339台のコンソール。 この巨大インフラは、化石資源が止まれば機能しない。

2026年イラン戦争の直撃： オイルショックにより電力・材料・物流費が同時に高騰。 透析施設のコスト構造が崩壊する。 患者一人あたりの年間費用は480万円から600万円超へ。 患者数は減少しているのに（337,414人→推計325,380人）、 総医療費は1.6兆円から2兆円規模へ上方爆発する。 人口動態の恩恵が、インフレに完全に食われる。

2026年度の診療報酬改定で、光熱水費の補填はわずか+0.09%。 オイルショックの現実とは絶望的に乖離している。 中小の透析クリニックは経営不能に陥り、 「透析難民」が発生するリスクが現実化する。

社会保険料30%——現役世代の限界

透析は氷山の一角だ。 問題は、日本の社会保障制度全体の構造にある。

現役世代が背負う負担：

健康保険料 → 約10%
厚生年金保険料 → 約18.3%
介護保険料 → 約1.8%
雇用保険料 → 約0.9%

合計 → 約31%（労使折半前の実質負担）

給与の3割以上が、働く前に消える。 しかも、この比率は毎年上がり続けている。 少子高齢化が進めば、40%、50%になる計算だ。

構造的に成立しない数学： 1970年：現役世代8.5人で高齢者1人を支えていた 2020年：現役世代2.1人で高齢者1人を支えている 2040年：現役世代1.5人で高齢者1人を支える予測 1.5人で1人を支える社会保障は、どんな制度設計でも破綻する。

年金制度——デスクワーク時代の遺物

現在の社会モデルは「65歳まで働く→年金で余生を過ごす」だ。

この制度が設計された時、前提は二つあった。 「余生は短い」と「人口は増え続ける」だ。

1960年代、日本人の平均寿命は65〜67歳。年金支給は55歳から。 余生は10年程度の想定だった。

いま、平均寿命は84歳。年金支給開始は65歳。 余生は20年。しかも、その20年間に医療費が膨大にかかる。

年金の構造的矛盾：

長寿化 → 年金支給期間が20年以上に
少子化 → 支える現役世代が減少
医療の高度化 → 高齢者の医療費が増大
社会保険料 → 現役世代の30%超

働く人が減り、遊ぶ人が増える → 数学的に破綻

しかし、問題はもっと根本的だ。 「65歳で引退する」という概念自体が、デスクワーク時代の産物ではないか。

デスクワーク社会の年金モデル： オフィスで座って働く → 運動不足＋加工食品 → 体を壊す 体を壊す → 医療に依存する → 医療費が膨張する 65歳で体が動かなくなる → 年金で養う → 現役世代が負担する これが「先進国」の社会モデルだった。

転換後の社会では、この前提が崩れる。

土地ベースの仕事がもたらす構造変化：
身体を使う仕事 → 運動が生活に組み込まれる
自分で育てた食物 → 加工食品への依存が減る
自然な食と運動 → 生活習慣病のリスクが下がる
→ 医療への依存が減る
→ 65歳で「引退」する必要がない
→ 年金への依存が減る

「病気を治す」のではなく「病気にならない」。 「年金で生活する」のではなく「年金が不要な暮らし」。 これは福岡正信の「何もしない農業」と同じ構造の逆転だ。

自然から離れるほど、生きるコストは上がる。 自然に近づけば、制度に頼る必要が減る。

アメリカ農業の構造的破綻

「日本の農業は非効率だ、アメリカのように大規模化すべきだ」 ——この主張は、アメリカ農業の構造を全く理解していない。

アメリカ農業の実態：

大規模モノカルチャー → 同じ作物を広大な土地に → 土壌微生物の多様性が消滅
化学肥料への完全依存 → 窒素・リン・カリウムを外部から投入 → 土壌は「培地」に退化
大量の農薬 → 除草剤（グリホサート等）→ 土壌微生物をさらに殺す
大型機械 → 土壌を圧縮 → 水と空気の循環を破壊
灌漑への依存 → オガララ帯水層の枯渇 → 回復に数千年

表土の喪失——取り返しのつかない損失： アメリカの農地は、過去150年で表土の約50%を失った。 表土1インチ（2.5cm）の形成には500〜1,000年かかる。 大規模農業は、数千年かけて蓄積された土壌資本を、 数十年で使い果たしている。 これは採掘であって、農業ではない。

オガララ帯水層——枯渇する地下水： アメリカ中西部の穀倉地帯を支える巨大地下水脈。 年間の汲み上げ量が、自然補給量の数倍〜数十倍。 カンザス州やテキサス州では、すでに水位が30m以上低下。 完全な回復には数千年〜数万年かかる。 アメリカの「食料安全保障」は、枯渇する地下水の上に建っている。

項目 アメリカ大規模農業 自然農法
--- --- ---
土壌 消耗品（採掘して使う） 生きたシステム（育てる）
肥料 外部依存（化石資源由来） 不要（微生物が供給）
水 地下水を採掘（枯渇する） 雨水＋土壌の保水力
農薬 必須（モノカルチャーの宿命） 不要（生態系バランス）
エネルギー 大量消費（機械＋輸送） 最小限（人力＋自然力）
期間 持続不可能（数十年で土壌崩壊） 持続可能（数千年の実績）
危機耐性 極めて脆弱（サプライチェーン依存） レジリエント（自立型）

アメリカ農業は「効率的」に見えるが、 それは土壌と地下水という資本を食い潰しているからだ。 銀行口座の残高を使い果たしながら「収入が多い」と言っているに等しい。

日本の本当の資源——水

「日本には資源がない」

これは日本で最も広く信じられている誤解だ。

日本には水がある。これが最大の天然資源だ。

水が豊かな国の構造的強み：

年間降水量 → 日本は約1,700mm（世界平均の約2倍）
水源 → 山岳地帯が国土の73%。天然の貯水・浄水システム
河川密度 → 極めて高い。どこでも水にアクセスできる
地下水 → 豊富で清浄
温暖湿潤気候 → 植物の成長に最適

水がなぜ最大の資源か： 石油は代替できる（太陽光、風力、核融合）。 水は代替できない。 農業には水が必要。 微生物には水が必要。 植物には水が必要。 人間には水が必要。 アメリカのオガララ帯水層は枯渇している。 中東は慢性的な水不足。 中国北部は深刻な水危機。 日本は、毎年空から1,700mmの水が降ってくる。 これを「資源がない」と言うのは、 石油とレアメタルしか「資源」と認識していないからだ。

水があれば、植物が育つ。 植物が育てば、微生物が育つ。 微生物が育てば、土壌が再生する。 土壌が再生すれば、食料が生まれる。 食料があれば、人間は生きられる。

日本の本当の資源循環：

豊富な雨水 → 多様な植物 → 土壌微生物 → 肥沃な土壌 → 食料
↑ ← ← ← ← ← ← ← ← ← ← ← ← ← ← ← ←（循環）← ↓

石油は掘れば減る。水は循環する。 これが「資源」の本質的な違いだ。

多様な植物と暮らしの知恵

日本列島は南北3,000kmにわたり、 亜熱帯から亜寒帯まで、驚くべき植物の多様性を持つ。

失われつつある知恵： どの野草が食べられるか。 どの植物が薬になるか。 どの木が建材になり、どの草が繊維になるか。 いつ種を蒔き、いつ収穫するか。 どの土地にどの作物が合うか。 この知恵は、数千年かけて蓄積されてきた。 しかし、スーパーマーケットとコンビニの時代に、 急速に失われている。

この知恵を残さなければならない。

化石資源が止まり、サプライチェーンが崩壊し、 スーパーの棚が空になった時、 生き残るのは「自分の土地で食べ物を作れる人」だ。 そのために必要なのは、最先端テクノロジーではない。 水と、植物と、暮らしの知恵だ。

福岡正信の思想——生きるためにどうするか

全ての構造分析は、一つの問いに収束する。

生きるためにどうするか。

福岡正信の自然農法は、この問いへの答えだ。

福岡正信の四つの原則の本質：

無耕起 → 土壌の構造を壊さない → 微生物が生きる
無肥料 → 外部依存をゼロにする → サプライチェーンに縛られない
無農薬 → 生態系を殺さない → 自然の防御力を活かす
無除草 → 光合成面積を最大化 → Light Farmingの原理

そしてChristine Jones博士のLight Farmingが、 この思想の科学的根拠を与えた。

Light Farming——福岡正信の科学的証明：

太陽光 → 光合成 → 植物が糖を作る
糖 → 根から土壌へ滲出（Liquid Carbon）
Liquid Carbon → 土壌微生物のエネルギー源
土壌微生物 → 植物に必要なミネラルを可溶化して供給
植物 → さらに光合成 →（正のフィードバックループ）

雑草も光合成する。草を残せば、Liquid Carbonの供給源が増える。
福岡正信の「無除草」は、Light Farmingの最大化戦略だった。

かなり違う世界にしなければならない

ここまでの議論を振り返る。

現在の社会の前提が、全て崩れる：
化石資源 → 枯渇する（第2章）
食料生産 → 化学肥料依存から脱却する（第3章）
エネルギー → 核融合でも素材問題は解決しない（第5章）
産業 → 軍需、IT、デスクワークが縮小する（第6〜8章）
人口分布 → 東京一極集中の理由が消える
貿易 → 自由貿易の前提が崩れる
医療・年金 → 都市型サラリーマン前提の制度が合わなくなる

個別の修正では済まない。 今とはかなり違う世界を設計しなければならない。

仕事が変わる。住む場所が変わる。貿易が変わる。医療と年金が変わる。 しかし、変わらないものもある。

水がある。植物がある。微生物がいる。土地がある。
日本の本当の資源は、最初から足元にあった。
