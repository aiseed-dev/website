## 原油がなければデータセンターは動かなくなる

AGI（汎用人工知能）が人類の未来を変える——
テック業界はそう言う。しかし、その前提を問う者は少ない。

AGIを動かすのはデータセンターだ。
そのデータセンターは何で作られているか。

:::highlight
**データセンターに不可欠な石油化学素材：**
基板（PCB） → エポキシ樹脂＋ガラス繊維（石油）。代替なし。
半導体パッケージング → エポキシモールド（石油）。代替なし。
フォトレジスト → 感光性樹脂（石油化学）。半導体製造の根幹。代替なし。
サーバー筐体 → ABS、ポリカーボネート（石油）。耐熱性・難燃性が必要。
ケーブル被覆 → PVC、ポリエチレン（石油）。難燃規格を満たすバイオ代替なし。
光ファイバー被覆 → アクリレート（石油）。光学精度が必要。代替なし。
冷却液ホース → 合成ゴム、PVC（石油）。耐薬品性が必要。
静電防止床材 → 石油系ビニル。静電気放電は機器破壊に直結。代替なし。
:::

これらは消耗品だ。サーバーは3〜5年で交換される。
ケーブルは劣化する。冷却系統は摩耗する。
データセンターは**継続的に石油化学素材を消費し続ける**施設だ。

:::chain
**AGIの物質的依存：**
AGIを動かす → データセンターが必要
→ データセンターには半導体が必要
→ 半導体製造にはフォトレジストが必要
→ フォトレジストには石油化学原料が必要
→ 石油化学原料にはナフサが必要
→ ナフサには原油精製が必要
→ **原油がなくなれば、半導体が作れない**
→ **半導体がなければ、AGIは動かない**
:::

AIが石油に代わる未来を語るなら、
まず「AIを動かすインフラ自体が石油に依存している」事実と向き合うべきだ。
電力だけの問題ではない。**素材の問題**だ。
再生可能エネルギーでデータセンターに電力を供給しても、
サーバーの基板、半導体のフォトレジスト、ケーブルの被覆——
これらは石油化学でしか作れない。

## データセンターの中の天才は何人必要か

仮に素材問題が解決し、AGIが実現したとしよう。
次の問いはこうだ——**データセンターの中の天才が必要な仕事は、どれだけあるか**。

AGIは言語の天才だ。情報の天才だ。分析の天才だ。
しかし、データセンターの中にいる。物理世界に出てこれない。

:::compare
| AGIにできる仕事 | AGIにできない仕事 |
| --- | --- |
| 書類を作る | 土壌を再生する |
| コードを書く | 植物を植える |
| 翻訳する | 菌根菌のネットワークを育てる |
| レポートを分析する | 壊れた水道管を直す |
| 戦略を立てる | 建物の断熱を改修する |
| 法律文書を作成する | 荒廃した森林を再生する |
| 医療画像を診断する | 患者の体に触れて看護する |
| 金融市場を予測する | 田んぼに水を引く |
:::

左側の仕事は全てオフィスの中、画面の中で完結する。
右側の仕事は全て物理世界で、人間の手と足が必要だ。

:::highlight
**構造的に見ると：**
世界の農地の33%が劣化している → 回復には人間の手が必要
化学肥料なしの野菜栽培法 → 研究から始めなければならない
都市インフラの大半が老朽化 → 分散型に再設計・施工が必要
生態系が世界中で劣化 → 森林・河川・湿地の物理的な回復作業が必要
これらは数十億人が数十年かけても終わらない規模の仕事だ。
:::

「AIに仕事を奪われる」と言っている人間は、
画面の中の仕事だけを「仕事」と定義している。
物理世界には、AIにはできない仕事が無限にある。

人間の仕事は減らない。増える。
ただし、それはオフィスの仕事ではなく、**土と水と生態系の仕事**だ。

### UBI（ベーシックインカム）が間違いである理由

「AIが仕事を奪うなら、ベーシックインカムで解決する」——
これも同じ問題設定の誤りの上に立っている。

しかし、UBIにはもっと根本的な問題がある。
**化石資源が無限にあることが前提になっている**。

:::highlight
**UBIの隠れた前提：**
金を配る → 人々が消費する → 経済が回る
しかし、消費される製品は何で作られているか。
食品包装 → プラスチック（石油）
衣類 → ポリエステル（石油）
家電 → 石油化学素材の塊
物流 → 軽油で動くトラック
**UBIで消費を回すとは、石油化学製品の消費を回すことだ。**
化石資源が有限である以上、この前提は成り立たない。
:::

:::chain
**AGI + UBIの論理（二重に間違い）：**
AIが仕事を奪う → 失業者が増える
→ 金を配る → 消費で経済を維持
→ しかし消費される製品は石油化学素材で作られている
→ 化石資源は有限 → **消費経済自体が持続しない**
:::

そして、化石資源の後に来るバイオ素材の製造には、**大量の人手が必要**だ。

:::highlight
**バイオ素材製造に必要な人的労働：**
微生物の培養 → 培養施設の建設・運営・管理に人手が必要
植物原料の栽培 → 農地の整備、栽培、収穫に人手が必要
菌糸体の生産 → 基質の準備、培養条件の管理に人手が必要
バイオマスの収集 → 農業廃棄物の収集・運搬は物理的労働
品質管理 → 生物由来の素材は品質のばらつきが大きい。人間の判断が必要
:::

石油化学プラントは少人数で大量生産できる。
バイオ素材は、生物を相手にするため、自動化が難しく、人手がかかる。
UBIで人を遊ばせている余裕はない。
**バイオ素材は生物を相手にする限り、永久に人手が必要だ。**
石油化学は一度プラントを作れば少人数で回せた。
バイオ素材はそうはいかない。微生物は生きている。植物は季節で変わる。
品質は毎回違う。これは「移行期の問題」ではなく、**生物を扱う限り永遠に続く構造**だ。

問題は「仕事がないこと」ではない。
問題は「やるべき物理的な仕事に価値が認められていないこと」だ。

## 過去のデータを大量に集めて意味があるのか

AGIのもう一つの根本的な問題。
現在のAIは、過去のデータを大量に集めてパターンを学習する。
しかし、**過去のデータで未来が見えるのか**。

:::highlight
**過去のデータが役に立たない領域：**
気候変動 → 過去のパターンが通用しない。前例のない異常気象が常態化している。
地政学 → 2026年イラン戦争を過去のデータから予測できたか。できなかった。
生態系の崩壊 → 閾値を超えると不可逆的に変化する。過去のデータに閾値は記録されていない。
土壌微生物 → 複雑系。個々の微生物の挙動からシステム全体の振る舞いを予測できない。
石油精製の停止 → 「起きたことがない」ことは、データに存在しない。
:::

過去のデータが豊富にある領域——金融市場、消費者行動、言語パターン——では、
AIは確かに強力だ。しかし、これらは全て**過去の延長線上に未来がある領域**だ。

:::chain
**AIが機能する条件：**
過去のパターンが未来にも当てはまる → AIは有効
過去に例のない変化が起きる → AIは無力
**今直面している問題：**
化石資源の枯渇 → 人類史上初
生態系の大規模崩壊 → 人類史上初
石油精製に依存した文明の終焉 → 人類史上初
→ **過去のデータに答えはない**
:::

福岡正信は50年前に、化学肥料依存の問題を見抜いた。
Christine Jones博士はLight Farmingの原理を発見した。
これらは過去のデータの分析ではなく、**構造を見抜く力**から生まれた。

AIが得意なのは「過去のパターンを高速で繰り返すこと」だ。
人間に必要なのは「過去にないパターンを見抜くこと」だ。
この二つは根本的に異なる能力であり、AGIは後者を代替できない。

## 現在のAIは既に使われている

AGIの議論の前に、現在のAIが何をしているかを見るべきだ。

:::highlight
**AIの実績：**
**軍事** — ウクライナ軍はAIをドローン制御、情報分析、標的選定に実戦投入している。AGIを待つまでもなく、現在のAIが戦場を変えている。
**ソフトウェア開発** — Claude自体がClaudeを使って開発されている。AIがAIのコードを書き、テストし、改善する。プログラマーの生産性は既に数倍になっている。
**構造分析** — このサイトの分析もAI（Claude）との共同作業で書かれている。石油精製の構造、化学肥料の依存関係、核融合炉の素材——人間が問いを立て、AIが情報を整理し、構造を可視化する。
:::

これらは全て、**現在のAI**——AGIではない——がやっていることだ。
現在のAIは十分に強力で、十分に実用的だ。
AGIを待つ必要はない。

## AGIは採算が取れるのか

「AGIが実現すれば全てが変わる」——
しかし、誰もAGIの採算について語らない。

:::chain
**AGIの経済構造：**
AGIを実現するには → 現在のAIよりはるかに大規模なデータセンターが必要
→ 大規模データセンターの建設費 → 数兆〜数十兆円
→ 運用費（電力、冷却、保守、半導体交換）→ 年間数千億〜数兆円
→ **この投資を回収するには、企業がいくら払うか**
:::

問いはこうだ——データセンターの中に天才を閉じ込めて、
過去の大量データから答えを出す。その答えに、企業はいくら払うか。

:::compare
| | 現在のClaude | AGI（仮想） |
| --- | --- | --- |
| 開発・運用コスト | 大規模だが管理可能 | 桁違いに大きい |
| できること | コード生成、分析、翻訳、文書作成 | 左記＋「より賢い」 |
| 企業が払う価値 | 月額数万〜数十万円/ユーザー | ？ |
| 追加で解決する問題 | — | **何があるか不明** |
:::

現在のClaudeで、コードを書き、分析をし、翻訳をし、文書を作成できる。
AGIが「より賢く」なったとして、企業が追加で払う金額はいくらか。
現在のAIで十分な仕事が大半だとすれば、
AGIの追加価値は、その莫大な追加コストに見合わない可能性がある。

:::highlight
**AGIの採算問題：**
開発費 → 数兆〜数十兆円（GoogleやMicrosoftの年間投資を見よ）
運用費 → 年間数千億〜数兆円（電力、冷却、半導体）
追加価値 → 現在のAIとの差分でしかない
**「現在のAIで十分」な領域が広がるほど、AGIの追加投資の回収は難しくなる**
:::

そしてこの全てが、石油化学素材で作られたデータセンターの中で動いている。
化石資源が有限である以上、AGIの経済モデルは二重に持続不可能だ。

:::quote
AGIは情報の世界を完成させるかもしれない。
しかし、AGIを動かすデータセンター自体が石油で作られている。
過去のデータに未来の答えはない。
現在のAIは既に十分に強力だ。
そして、壊れた物理世界を直すのは、人間の手でしかできない。
:::
