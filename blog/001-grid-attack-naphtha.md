## 何が起きているか

トランプ政権がイランの電力網を攻撃した。イランは報復として湾岸の石油施設を攻撃し、ホルムズ海峡に機雷を設置した。

多くの人はこれを「中東の戦争」として見ている。

違う。**これは日本の食料危機だ。**

## ホルムズ海峡が閉まると何が止まるか

日本の原油輸入の約9割は中東からだ。そのほぼ全てがホルムズ海峡を通過する。

:::chain
ホルムズ海峡封鎖 → 原油輸入停止 → 製油所の原料がなくなる → ナフサ・ガソリン・灯油・軽油　全て停止
:::

[第2章「化石資源と素材」](/insights/fossil-materials/)で説明した通り、原油の蒸留は「選べない」。ガソリンだけ作ることも、ナフサだけ作ることもできない。原油が来なければ、全てが止まる。

## ナフサが止まると肥料が止まる

ナフサは石油化学の原料だ。プラスチック、合成繊維、医薬品——現代文明の素材は全てナフサから始まる。

そしてナフサ（または天然ガス）から作られるものの中に、**アンモニア**がある。

:::chain
ナフサ / 天然ガス → 水素 → アンモニア → **窒素肥料**
窒素肥料 → 世界の食料生産の約50%を支えている
:::

日本は肥料原料もほぼ全量輸入に依存している。

- **窒素肥料**の原料：天然ガス / ナフサ（中東依存）
- **リン酸肥料**の原料：リン鉱石（中国・モロッコから輸入）
- **カリウム肥料**の原料：カリ鉱石（カナダ・ロシアから輸入）

ホルムズ海峡の封鎖は、窒素肥料の原料供給を直撃する。

:::highlight
ホルムズ海峡封鎖 → 原油・天然ガス停止 → ナフサ停止 → アンモニア停止 → 窒素肥料停止

これは「ガソリンが高くなる」問題ではない。**食料生産が止まる**問題だ。
:::

## 湾岸施設への攻撃——生産自体が止まる

さらに深刻なのは、イランが湾岸の石油施設を攻撃していることだ。

ホルムズ海峡の封鎖は「通れない」問題だが、施設への攻撃は「作れない」問題だ。海峡の機雷は掃海できるが、**破壊された精製施設の復旧には年単位の時間がかかる**。

:::highlight
**海峡封鎖は解除できる。施設破壊は復旧に3〜5年かかる。**

つまり戦争が終わっても、供給はすぐには戻らない。
:::

## 日本の脆弱性

日本の数字を並べる。

- 原油の中東依存度：**約90%**
- ナフサの大半を**輸入に依存**
- 肥料原料：**ほぼ全量輸入**
- 食料自給率：**38%**（カロリーベース）

この構造は、ホルムズ海峡という一つのチョークポイントに全てが集中していることを意味する。

:::highlight
**日本の食料は、ホルムズ海峡に依存している。**

中東の戦争は、日本の食卓の問題だ。
:::

## 構造分析シリーズとの接続

[第1章](/insights/climate-mistake/)で指摘した通り、化石資源の枯渇は気候変動より先に来る。

[第2章](/insights/fossil-materials/)で示した通り、原油の蒸留は選べない。止まれば全てが止まる。

[第3章](/insights/agriculture/)で分析した通り、化学肥料が消えれば慣行農業は成り立たない。

[第11章](/insights/regulation-redesign/)で述べた通り、脱炭素が硫黄供給を止め、リン酸肥料も危機に陥る。

イラン戦争は、この構造を**今この瞬間に現実にしている**。

化石資源の枯渇は数十年後に来る。しかしホルムズ海峡の封鎖は**今日すでに起きている**。構造分析シリーズが描いた未来が、予想より早く来た。

## 自然農法だけが影響を受けない

化学肥料に依存しない農業だけが、この構造的リスクから自由だ。

福岡正信の自然農法、Christine Jones博士のLight Farming。不耕起、無肥料、無農薬。土壌微生物との共生による窒素固定。

ホルムズ海峡が閉まっても、土壌微生物は止まらない。

:::highlight
**構造分析の結論は常に同じ場所に収束する。**

自然から離れるほど、生きるコストは上がる。イラン戦争は、そのコストが一瞬で跳ね上がることを証明した。
:::
