## あなたの生活のほぼ全てが石油でできている

朝起きる。ポリエステルのシーツから体を起こす。
プラスチックの歯ブラシで歯を磨く。
ナイロンの服を着る。合成ゴムの靴を履く。
スマートフォンのポリカーボネート筐体を手に取る。
アスファルトの道路を歩く。
食品をポリエチレンの袋から取り出す。
ポリプロピレンの容器に入った弁当を食べる。

この日常のどこに「石油」があるか。全部だ。

:::chain
**石油から作られる日常の素材：**
ナフサ → エチレン → ポリエチレン（レジ袋、ラップ、容器）
ナフサ → プロピレン → ポリプロピレン（食品容器、自動車部品）
ナフサ → ベンゼン → ナイロン、ポリウレタン（衣類、断熱材）
ナフサ → スチレン → ポリスチレン（発泡スチロール、食品トレー）
ナフサ → PET（ペットボトル、ポリエステル繊維）
ナフサ → 塩ビ（配管、窓枠、電線被覆）
重質留分 → アスファルト（道路）
重質留分 → 潤滑油（あらゆる機械の動作）
副産物 → パラフィン（ろうそく、化粧品、医薬品コーティング）
:::

プラスチックは「環境に悪い使い捨て製品」として語られることが多い。
しかし、それは問題の表面にすぎない。
本当の問題は、現代文明のインフラそのものが化石資源の素材機能に依存していることだ。

## プラスチック問題の本質は「代替不能性」

「プラスチックをやめよう」「脱プラスチック」——
この議論は、プラスチックを「選択肢の一つ」として捉えている。
紙に変えればいい。竹に変えればいい。ガラスに変えればいい。

しかし、構造的に見れば、プラスチックが代替不能な用途が大量にある。

:::compare
| 用途 | なぜプラスチックなのか | 代替の現実 |
| --- | --- | --- |
| 医療器具（注射器、チューブ、血液バッグ） | 滅菌可能、軽量、使い捨てで感染防止 | ガラスは重く割れる。金属は高コスト。現実的な代替なし |
| 食品包装 | 気密性、軽量、低コスト、食品との非反応性 | 紙は水に弱い。ガラスは重く輸送コスト増。部分的代替のみ |
| 電線被覆 | 絶縁性、柔軟性、耐候性 | ゴムは劣化が速い。代替なし |
| 水道管・下水管 | 腐食しない、軽量、長寿命 | 金属管は腐食する。陶管は重く脆い。大規模代替は非現実的 |
| 自動車部品 | 軽量化（燃費向上）、成形の自由度 | 金属で全て作ると重量・コスト増大 |
| 半導体のフォトレジスト | 微細パターン形成に不可欠 | 代替なし。半導体製造が停止する |
:::

「脱プラスチック」は、レジ袋やストローの話をしている。
しかし、医療器具、電線、水道管、半導体のプラスチックをやめたら、
文明が止まる。

:::highlight
**プラスチック問題の構造：**
表面の問題：使い捨てプラスチックの環境汚染（レジ袋、ストロー）
本質の問題：現代文明のインフラがプラスチックなしでは機能しない
見えていない問題：そのプラスチックの原料（ナフサ）は石油精製から来る
:::

## 大企業が存在できるのは化石資源があるから

現代の大企業——Amazon、Apple、Toyota、Walmart——は、
化石資源のインフラの上に成り立っている。

:::chain
**大企業を支えるインフラの化石資源依存：**
**物流：**軽油（トラック・船舶）+ アスファルト（道路）+ プラスチック（梱包材）
**製造：**ナフサ（部品素材）+ 潤滑油（機械動作）+ 合成ゴム（シール・ベルト）
**通信：**塩ビ（電線被覆）+ フォトレジスト（半導体）+ 光ファイバー被覆
**食品：**ポリエチレン（包装）+ 硫黄（肥料）+ ナフサ（農薬原料）
**建設：**アスファルト + 塩ビ配管 + 断熱材（ポリウレタン）+ 塗料
:::

Amazonの翌日配送は、アスファルトの道路とプラスチックの梱包材がなければ成立しない。
Appleの製品は、半導体のフォトレジストとポリカーボネートの筐体がなければ作れない。
Toyotaの車は、合成ゴムのシールと潤滑油がなければ動かない。

大企業は「テクノロジー企業」や「製造企業」を自称するが、
構造的には「化石資源の加工業」だ。
化石資源の素材機能がなくなれば、これらの企業のビジネスモデルは崩壊する。

## 全車EV化は無謀である

「全ての自動車を電気自動車にすれば、石油はいらなくなる」——
この発想が、素材の構造を完全に無視している。

### ガソリンは減らせる。しかし軽油はどうにもならない。

乗用車のEV化は合理的だ。ガソリン需要を減らすことはできる。
しかし、**軽油は別の話だ**。

:::highlight
**電化が極めて困難な領域：**
大型トラック（長距離輸送）→ バッテリーの重量で積載量が激減。充電時間が物流を止める。
船舶 → 大型コンテナ船の電化は物理的に不可能。重油・軽油が唯一の現実解。
建設機械 → 高負荷・長時間稼働。電動化は小型機のみ。
農業機械 → 圃場に充電インフラはない。軽油で動くしかない。
:::

そして、ナフサのために原油を精製すれば、軽油は**必ず**同時に出てくる。

:::chain
**精製の構造的事実：**
プラスチック、医療器具、肥料原料にナフサが必要
→ ナフサは原油精製でしか得られない
→ 原油を精製すれば、ガソリン・軽油・重油が同時に出てくる
→ 素材が必要な限り、石油精製は止められない
→ 精製が止められないなら、**軽油はどうしても出てくる**
→ トラック・船舶・建設機械・農機がそれを使う——これが最も合理的だ
:::

ガソリンについては、乗用車のEV化で需要を減らすことに意味はある。
しかし、軽油は二重の理由で消えない——
**使う側が電化できない**。そして**精製すれば必ず出てくる**。

:::highlight
**構造的事実：**
現在、石油精製のコストは86%のエネルギー用途（ガソリン・軽油）の売上で賄われている。
残り14%の化学原料（ナフサなど）は、いわば「ついで」に安く供給されている。
燃料需要が大幅に減れば、素材用途のためだけに石油を精製することになる。
精製コストは現在の何倍にもなる。
ナフサの価格が跳ね上がり、プラスチック、医療器具、肥料——全てが高騰する。
:::

問題は「ガソリン車かEVか」ではない。
**素材の代替（バイオ素材）がどこまで進んでいるか**だ。
バイオ素材がナフサを代替できるようになって初めて、
石油精製を縮小する選択肢が生まれる。順序が逆なのだ。

## 化石資源の後に来るもの——微生物と植物

化石資源が枯渇するか、素材の代替が進んで精製が縮小するか——
いずれにせよ、ナフサベースの素材が手に入らなくなる時代が来る。

その時、素材はどこから来るのか。
答えは二つしかない。**微生物と植物**だ。
微生物由来の素材
**バイオプラスチック**——微生物が糖を発酵させてPHA（ポリヒドロキシアルカノエート）を生産。生分解性。
**バイオ繊維**——細菌セルロース。ナタデココと同じ原理。
**バイオ接着剤**——菌糸体ベースの接着剤。
**発酵化学品**——エタノール、乳酸、コハク酸。化学原料として使用可能。
植物由来の素材
**セルロース**——木材、麻、竹から。紙、繊維、建材。
**リグニン**——木材の副産物。接着剤、炭素繊維の原料候補。
**天然ゴム**——ゴムの木から。合成ゴムの代替。
**植物油**——亜麻仁油、桐油。塗料、潤滑剤。
**デンプン**——生分解性プラスチックの原料。
菌糸体（マイセリウム）
キノコの根に当たる菌糸体は、素材革命の鍵を握る。
断熱材、梱包材、建材、皮革代替品として既に実用化が始まっている。
農業廃棄物を基質として成長し、生分解性。
発泡スチロールの直接的な代替候補。
藻類
微細藻類はバイオ燃料だけでなく、バイオプラスチック、
食品添加物、化粧品原料、飼料を生産できる。
CO2を吸収しながら成長し、耕作地を必要としない。
水と光とCO2があれば、どこでも培養可能。

## バイオ素材の正直な限界

ここで正直に言わなければならないことがある。
バイオ素材は、現時点では石油化学製品の代替にならない。

:::highlight
**バイオ素材の構造的問題：**
**コスト** — バイオプラスチック（PHA）の製造コストは石油由来プラスチックの3〜5倍。大量生産してもコスト差は簡単には縮まらない。
**スケール** — 世界のプラスチック生産量は年間約4億トン。バイオプラスチックは全体の1%未満。100倍のスケールアップは10〜20年では不可能。
**性能** — 耐熱性、強度、耐久性で石油由来に劣る用途が多い。医療器具や半導体のフォトレジストの代替は現時点で存在しない。
**土地** — バイオマス原料の栽培には農地が必要。食料生産と競合する。
**エネルギー** — 発酵・精製プロセスに大量のエネルギーが必要。「脱石油」のためにエネルギーを大量消費するパラドックス。
:::

これは「やがて解決する技術的課題」ではない。
物理的・生物学的な制約だ。
微生物の増殖速度は化学プラントの処理速度に及ばない。
植物の成長速度は需要の成長速度に追いつかない。

## それでもやるしかない

しかし、化石資源は有限だ。
石油は掘れば減る。天然ガスも掘れば減る。
100年後、200年後、必ず枯渇する。
ホルムズ危機のような地政学リスクで、突然止まることもある。

バイオ素材が石油化学に劣ることは事実だ。
しかし、石油化学がいつまでも使える前提で暮らすことはできない。
「劣るからやらない」のではなく、
「劣るけど今から始めなければ間に合わない」。

:::chain
**移行の時間軸：**
バイオ素材の技術開発 → 10〜20年
製造スケールの拡大 → 20〜30年
社会インフラの転換 → 30〜50年
石油が確実に使える残り時間 → **不明**（地政学リスクで突然止まりうる）
:::

だからこそ、今ある化石資源は大切に使わなければならない。

## 化石資源は大切に使え

化石資源が有限であることを理解すれば、
今の使い方がいかに愚かかが分かる。

:::highlight
**現在の使い方：**
石油の86%をエネルギーとして**燃やしている**。
燃やせばCO2になり、二度と素材には戻らない。
残り14%が化学原料（ナフサなど）として素材に使われている。
86%を燃やす余裕は、本来ないはずだ。
:::

しかし、ここに構造的な矛盾がある。

:::highlight
**精製の構造的制約：**
原油は精製すると、ナフサ・ガソリン・軽油・重油が**同時に**出てくる。
ナフサだけを取り出すことはできない。
現在の量の化学原料（ナフサ）が必要な限り、原油の精製は続く。
精製すれば、燃料も必ず出てくる。
つまり、「石油を燃やすのをやめろ」は、精製の構造上、単純には成り立たない。
:::

ではどうするか。

**短期的には、精製で出てくる燃料を使いながら、需要の構造を変える。**
使い捨てプラスチックの浪費、過剰包装、短寿命の製品設計——
これらを減らせば、必要なナフサの量が減り、精製量が減り、燃料の副産物も減る。

**中長期的には、バイオ素材でナフサ需要そのものを減らす。**
ナフサ需要が減れば、精製量が減り、燃料の副産物も減る。
そこに再生可能エネルギーの拡大を組み合わせれば、
石油精製を段階的に縮小できる。

再生可能エネルギーへの転換は、「環境のため」だけではない。
**有限な化石資源を素材として長く使うため**でもある。
しかし、素材需要がある限り精製は止められない——
この構造的制約を無視した「脱石油」は空論だ。

:::compare
| 戦略 | 短期（5年） | 中期（20年） | 長期（50年） |
| --- | --- | --- | --- |
| エネルギー | 再エネ拡大。燃料副産物の依存を減らす | 再エネ主力化 | 精製量の大幅縮小 |
| 素材 | 使い捨て削減。ナフサ需要を減らす | バイオ素材のスケールアップ | バイオ素材が主力 |
| 土壌 | 再生農業の開始 | 土壌微生物の回復 | 生物学的インフラの確立 |
| 備蓄 | ナフサ・肥料の戦略備蓄を新設 | 備蓄量の拡大 | 備蓄不要（自給体制） |
:::

## 福岡正信が見ていた未来

福岡正信は1970年代に「わら一本の革命」を書いた。
石油危機の直後だ。
しかし、彼が批判したのはエネルギー依存ではなく、
化学肥料と農薬への依存——つまり、石油の素材機能への依存だった。

:::quote
自然農法は、石油文明が終わった後の農業モデルではない。
石油文明が始まる前から存在していた農業モデルだ。
微生物と植物が素材を供給し、人間はその中で暮らしていた。
それが本来の姿であり、戻るべき場所だ。
:::

リジェネラティブ農業は、単に「化学肥料を使わない」農業ではない。
土壌微生物を育て、菌根菌のネットワークを再構築し、
植物と微生物の共生系を回復させる——
これは、化石資源後の世界で素材を供給するシステムの基盤でもある。

## 構造的に見える未来

化石資源後の世界は、「不便になる」世界ではない。
素材の供給源が変わる世界だ。

:::chain
**素材供給の転換：**
石油精製 → ナフサ → プラスチック → 大量生産・大量廃棄
微生物発酵 → バイオポリマー → 生分解性素材
植物栽培 → セルロース・リグニン → 繊維・建材
菌糸体培養 → マイセリウム → 梱包材・断熱材
藻類培養 → バイオオイル → 化学原料
:::

この転換には、膨大な量の「生物学的インフラ」が必要だ。
微生物の培養施設、植物の栽培システム、菌糸体の生産拠点——
これらは全て、土壌の健康と生物多様性の上に成り立つ。

つまり、化石資源後の世界の準備は、
土壌の再生から始まる。

:::highlight
**結論：**
現代文明の便利さの正体は、化石資源から作られた素材だ。
エネルギーは代替できるが、素材の代替には微生物と植物が必要になる。
大企業も、日常の便利さも、化石資源の素材機能の上に成り立っている。
化石資源後の世界に備える唯一の方法は、
微生物と植物を育てる力を取り戻すこと——
すなわち、土壌を再生することだ。
:::
